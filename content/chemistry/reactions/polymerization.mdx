# 聚合反应

你手边几乎所有东西——手机外壳、矿泉水瓶、衣服纤维、轮胎、连同你眼角膜上的隐形眼镜——都是同一类化学反应的产物：**聚合**。

聚合，就是把成千上万个微小的分子（**单体**）像串珠子一样首尾相连，长成一条巨大的链状分子（**聚合物**，俗称高分子）。

正是这个反应，把人类带进了"塑料时代"，也把"会降解的塑料垃圾"这个问题留给了未来。

## 破除误解：塑料不是"一种东西"，分子也不是无限长

很多人把"塑料"当成单一材料。其实塑料是一个庞大家族：聚乙烯（保鲜膜、塑料袋）、聚丙烯（饭盒）、聚氯乙烯（PVC 管）、聚酯（饮料瓶、衣服）、尼龙……

它们性质天差地别，唯一共同点是都由小分子聚合而成。

另一个误解：以为聚合物分子"想多长就多长，是真正的巨型分子"。实际上每条链长度并不整齐，一份聚合物是**不同长度链的混合物**，化学家用"平均分子量"和"分散度"来描述它。

链的长短分布，直接决定材料是软是硬、是脆是韧。

还有人以为天然界没有高分子。恰恰相反：淀粉、纤维素、蛋白质、DNA、天然橡胶，全都是大自然早就玩熟的聚合物。

人类只是后来学会了人工合成。

## 反应机制：两条把小分子接成长链的路

把单体连成长链，主要有两条化学路线。

**第一条：加聚（加成聚合）。** 单体里含有不饱和的碳碳双键（C=C）。

在引发剂或催化剂作用下，双键被"打开"，腾出两只手分别去拉住前后的单体，于是一个接一个连成长链，过程中**没有任何小分子被排出**——原子全部保留在链里。聚乙烯就是这么造的：

```
n CH2=CH2 → —[CH2—CH2]n—
```

成千上万个乙烯（CH₂=CH₂）首尾相接，变成聚乙烯。

这往往是**链式反应**：一个活性中心（自由基或离子）不断吞入新单体快速增长，直到终止。

**第二条：缩聚（缩合聚合）。** 单体两端带有能反应的官能团（如 —COOH 和 —OH）。它们两两结合时，每连一次就**挤出一个小分子**（通常是水）。涤纶（聚酯）和尼龙都靠它造：

```
n HOOC—R—COOH + n HO—R'—OH → —[OC—R—CO—O—R'—O]n— + (2n−1) H2O
```

加聚像拉链一次拉好，干净利落；缩聚像两端各带钩子的链节逐对扣合，每扣一次掉一滴水。

两条路造出的高分子结构与性能也因此不同。

## 现实意义与应用：塑料如何重塑世界

聚合反应几乎重新定义了二十世纪的物质生活。

- **包装与日用**：聚乙烯、聚丙烯轻、便宜、不漏水，是塑料袋、瓶、桶的主力。
- **纤维与衣物**：尼龙（1935 年杜邦合成）、涤纶让衣服结实、易洗、不缩水，取代了大量天然纤维。
- **工程与建筑**：PVC 管道、有机玻璃、环氧树脂、聚氨酯泡沫，撑起了现代基建与隔热。
- **医疗**：可吸收手术缝线、人工血管、隐形眼镜、药物缓释微球，都是定制聚合物。
- **高科技**：芳纶（凯夫拉）做防弹衣，导电聚合物用于柔性屏，性能聚合物甚至上了飞机和火箭。

聚合反应的可贵之处，是能"按需设计"：调整单体、链长、支化和交联，就能精确定制软硬、透明度、耐热性。

几乎想要什么样的材料，都能这样造出来。

## 代价与争议：白色污染与微塑料

塑料的便宜耐用，反过来成了它最大的麻烦。

正因为聚合物链稳定、不易被微生物分解，丢弃的塑料能在环境里存留**数百年**。

它们破碎成肉眼难见的**微塑料**，已遍布海洋、土壤、饮用水，甚至进入人体血液和胎盘，长期健康影响仍在研究中。

塑料生产本身高度依赖石油，是化石资源消耗和碳排放的一环；焚烧某些塑料（如 PVC）还会释放有毒物质。

应对之道有几条：发展**可降解聚合物**（如聚乳酸 PLA，可在堆肥中分解）、提高回收率、用[[catalysis-reaction]]开发能把废塑料"解聚"回单体的化学回收技术。但目前全球塑料的实际回收率仍很低，这是一道远未解决的难题。

## 跨域连接

- **生命科学**：生命本身就建立在聚合之上——蛋白质是氨基酸的缩聚链，DNA 是核苷酸的长链，纤维素和淀粉是葡萄糖的聚合物。细胞用酶（生物催化剂）精确控制链长与序列，比工业聚合精密得多。
- **医学**：可降解缝线、人工关节、药物载体都是医用高分子；微塑料对健康的影响是当前公共卫生研究热点。
- **地球科学**：塑料污染已成为一种"人类世"的地层标志，未来的地质学家或许能从岩层里读出塑料时代。
- **化学**：聚合的反向是[[organic-synthesis]]中精确搭建单个分子；而许多单体（如乙烯）由石油裂解或[[fermentation]]生产的生物乙醇脱水而来。
