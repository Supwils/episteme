# 酸碱中和

抗酸药可以消耗一部分胃酸，工业废水也常要在排放前调节酸碱度。这些场景都利用酸与碱之间的反应，但绝不能据此推导出"接触了酸就涂碱、接触了碱就涂酸"。

蜂蜇、化学灼伤和误服都不是课堂烧杯。蜂毒成分复杂，外涂小苏打并不能靠简单中和可靠处理；浓酸浓碱接触皮肤时互相中和还会放热。现实急救应遵循当地医疗或毒物控制机构指引，化学品皮肤暴露通常首先持续用流动清水冲洗。

**酸碱中和**描述的是酸碱反应达到特定化学计量关系的过程，不是两种危险物质自动变得无害。

它是初中化学就出现的反应，却支撑着从制药、农业到人体血液稳定的庞大体系。看似简单，深挖下去全是讲究。

## 破除误解：中和不等于"变成纯净水"

一个常见误解是：酸加碱，恰好抵消，就一定得到中性的、pH=7 的纯水。

这只在"强酸配强碱"且用量精确相等时才成立。

如果是**弱酸配强碱**（比如醋酸加氢氧化钠），当量点附近生成的共轭碱会与水反应，使溶液偏碱性；**强酸配弱碱**则通常偏酸性。原因是生成物参与新的酸碱平衡。

所以"中和到底"并不总是 pH=7。

另一个误解：以为"酸性 = 危险，碱性 = 安全"。其实强碱（如烧碱 NaOH）对皮肤和眼睛的腐蚀，往往比同浓度的酸更严重——它会让蛋白质和脂肪溶解。

酸碱的危险来自浓度和强度，与"是酸还是碱"无关。

## 反应机制：H⁺ 与 OH⁻ 结合成水

按照 Arrhenius 的经典定义：酸增加水溶液中的氢离子，碱增加氢氧根离子。裸露 H⁺ 不会在水里独自存在，常用 H⁺ 是对水合质子（如 H₃O⁺）的简写。强酸与强碱在稀水溶液中的净反应可写为：

```
H3O+ + OH- → 2H2O
```

这条净离子方程式解释了许多强酸—强碱反应为何有相近的摩尔反应焓。对弱酸、弱碱、浓溶液或非水溶剂，电离、溶剂化和稀释放热都会加入能量账本，不能把一个约数当作所有中和的固定热量。

在足够稀的水溶液中，强酸与强碱每生成一摩尔水的中和焓常接近 $-57\ \mathrm{kJ\,mol^{-1}}$。实验值仍受温度、浓度和量热条件影响。

完整地看一个例子——盐酸和氢氧化钠：

```
HCl + NaOH → NaCl + H2O
```

酸提供的水合质子和碱提供的 OH⁻ 生成水，Na⁺ 与 Cl⁻ 作为旁观离子留在溶液中。对这一类 Arrhenius 反应，常可概括为"盐 + 水"。

更现代的 Brønsted–Lowry 定义把视角拓宽：酸是**质子（H⁺）的给予者**，碱是质子的接受者。中和，就是一次质子从酸到碱的转移。

这个定义能解释为什么氨气（NH₃，没有 OH⁻）也是碱——它能接住一个质子变成 NH₄⁺。

扩展以后，产物不一定含水。HCl 气体与 NH₃ 形成 NH₄Cl，就是质子转移却没有水生成；碳酸盐与酸反应还会放出 CO₂。把"任何酸碱反应都生成盐和水"写成定律，会把入门特例误当成定义。

## 强弱、浓稀与 pH 是三件事

**强酸/弱酸**描述在指定溶剂中的电离平衡，常用 $K_a$ 或 $pK_a$ 比较；**浓/稀**描述单位体积里有多少物质。稀盐酸可以比浓醋酸具有更低或更高的 pH，必须给出实际浓度与平衡条件。

pH 更严格地与氢离子**活度**相关：

$$
\mathrm{pH}=-\log_{10}a_{\mathrm{H^+}}
$$

在足够稀的溶液里，活度可近似为浓度；离子强度高时，离子间相互作用使两者偏离。pH=7 代表中性也只在约 25°C 的水中成立，因为水的离子积随温度变化。中性真正表示 $a_{\mathrm{H^+}}=a_{\mathrm{OH^-}}$，不永远对应数字 7。

## 缓冲不是把酸碱一次性消灭

缓冲液通常含弱酸及其共轭碱，或弱碱及其共轭酸。加入少量强酸时，共轭碱接住质子；加入少量强碱时，弱酸提供质子，因此 pH 变化被压小。

在适当稀溶液近似下，Henderson–Hasselbalch 关系写为：

$$
\mathrm{pH}=\mathrm{p}K_a+\log_{10}\frac{a_{\mathrm{base}}}{a_{\mathrm{acid}}}
$$

它说明缓冲能力有上限，也通常在酸碱两种形态数量相近时最有效。超过容量后，pH 仍会迅速变化；缓冲不是无限中和器。

## 滴定：用中和来"称量"看不见的酸

既然中和反应在 H⁺ 和 OH⁻ 恰好相等时完成，化学家就用它来精确测定未知溶液的浓度——这就是**滴定**。

做法是：把已知浓度的碱（标准液）一滴一滴加入未知浓度的酸中，同时用**指示剂**（如酚酞，遇碱变红）或 pH 计盯着。

当滴定剂与待测物按反应式达到**当量点**，指示剂或仪器信号发生明显变化，记录体积即可反推浓度。对多元酸、弱酸或非 1:1 反应，当量关系不能简写成 H⁺ 与 OH⁻ 数目直接相等。

滴定是分析化学最基本、最可靠的定量手段之一，至今仍用于测食品酸度、药品纯度、水质硬度。

## 现实意义与应用

中和反应几乎渗透进每个角落：

- **医药**：部分抗酸药用碳酸盐或氢氧化物消耗胃酸，但剂量、钠负荷、肾功能和药物相互作用都会影响选择。
- **农业**：酸性土壤撒石灰（CaO/CaCO₃）中和后才适合多数作物；这是延续千年的农艺。
- **工业废水**：排放前必须把强酸或强碱废液中和到接近中性，否则污染水体。
- **人体血液**：碳酸氢盐、蛋白质、磷酸盐缓冲与肺排 CO₂、肾调节酸碱共同维持细胞外液 pH。缓冲负责分钟内抵抗变化，器官调节负责持续恢复，不能只说成一场中和反应。

## 代价与争议：腐蚀、热量与缓冲的脆弱

中和虽是"温和化解"，却暗藏风险。

强酸强碱混合时**剧烈放热**，工业上若一次性大量混合可能沸腾喷溅，必须缓慢、分批操作。

处理浓酸浓碱本身就有强腐蚀危险，需防护装备。

"被酸烧伤就用碱中和"这个民间急救法其实**危险**——中和放热会造成二次热灼伤，正确做法是先用大量清水冲洗稀释。

在环境层面，海洋正在经历一场缓慢的"中和危机"：大气 CO₂ 溶进海水生成碳酸，正持续降低海水 pH（海洋酸化），威胁珊瑚与贝类的碳酸钙外壳。

这提醒我们，地球尺度的酸碱平衡同样脆弱。

## 跨域连接

- **生命科学**：酶只在狭窄的 pH 范围内有活性，人体靠缓冲系统维持血液中性；胃靠盐酸（pH≈1.5）杀菌和消化，肠道再用碱性肠液中和，这是一场精密的酸碱接力。
- **地球科学**：海洋酸化、酸雨对石灰岩的侵蚀（生成可溶碳酸氢钙，再沉积成钟乳石），都是大自然里的中和与逆中和，与[[precipitation-reactions]]紧密相连。
- **医学**：酸中毒或碱中毒的治疗首先针对通气、循环、肾功能、代谢或中毒原因；是否使用碱性药物取决于具体病因和严重度，不是看到低 pH 就机械"中和"。
- **化学**：中和的逆向思路——用电能拆解盐水重新得到酸和碱——见[[electrolysis]]；而酸的另一来源是含硫燃料的[[combustion]]。
