## 定理陈述

**诺特定理**（Noether's Theorem）建立了对称性与守恒律之间的深刻联系。

设物理系统的拉格朗日量 $L(q, \dot{q}, t)$ 在某个连续变换下不变（即具有连续对称性），则存在一个对应的守恒量。

更精确地：若 $L$ 在无穷小变换 $q \to q + \varepsilon\delta q$ 下不变（$\delta L = 0$），则

$$Q = \frac{\partial L}{\partial \dot{q}} \cdot \delta q$$

是一个守恒量，即 $\frac{dQ}{dt} = 0$。

经典对应关系：
| 对称性 | 守恒律 |
|--------|--------|
| 时间平移不变性 | 能量守恒 |
| 空间平移不变性 | 动量守恒 |
| 空间旋转不变性 | 角动量守恒 |
| 规范对称性 | 电荷守恒 |

## 直觉理解

诺特定理说的是：**每一种对称性都对应一条守恒律**。

为什么能量守恒？因为物理定律不随时间改变——今天的物理定律和明天的一样。如果物理定律每天都在变，能量就不守恒了。

为什么动量守恒？因为物理定律不随空间位置改变——这里的物理定律和那里的一样。如果物理定律在不同地方不同，动量就不守恒了。

为什么角动量守恒？因为物理定律不随方向改变——空间是各向同性的。

诺特定理将看似无关的两个概念——对称性（几何性质）和守恒律（物理性质）——统一为同一枚硬币的两面。这是理论物理学中最深刻、最美丽的洞察之一。

## 证明思路

### 经典力学版本

设拉格朗日量 $L(q, \dot{q}, t)$ 在变换 $q \to q + \varepsilon\delta q$ 下不变。由最小作用量原理 $\delta S = 0$（其中 $S = \int L\,dt$），考虑无穷小变换：

$$\delta S = \int \left(\frac{\partial L}{\partial q}\delta q + \frac{\partial L}{\partial \dot{q}}\delta\dot{q}\right)dt = 0$$

由欧拉-拉格朗日方程 $\frac{d}{dt}\frac{\partial L}{\partial \dot{q}} = \frac{\partial L}{\partial q}$，化简得：

$$\delta S = \int \frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}}\delta q\right)dt = 0$$

因此 $\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}}\delta q\right) = 0$，即 $Q = \frac{\partial L}{\partial \dot{q}}\delta q$ 是守恒量。

### 场论版本

在经典场论中，拉格朗日密度 $\mathcal{L}(\phi, \partial_\mu\phi)$ 在变换 $\phi \to \phi + \varepsilon\delta\phi$ 下不变，对应的守恒流为

$$j^\mu = \frac{\partial \mathcal{L}}{\partial(\partial_\mu\phi)}\delta\phi$$

满足 $\partial_\mu j^\mu = 0$（连续性方程），对应的守恒荷为 $Q = \int j^0 d^3x$。

## 历史背景

**艾米·诺特**（Emmy Noether，1882—1935）是德国数学家，被誉为"抽象代数之母"。诺特定理发表于 1918 年。

诺特在哥廷根大学工作时，因性别歧视无法获得正式教职。**爱因斯坦**和**希尔伯特**为她的任命据理力争，但未能成功。希尔伯特曾愤怒地说："我不明白性别为什么是一个论据——我们毕竟是数学系，不是洗浴俱乐部。"

诺特的贡献远不止这一定理。她开创了环论和理想理论的抽象方法，发展了"诺特环""诺特模"等概念。她的学生遍布美国（她于 1933 年因纳粹迫害移居美国）。

诺特定理在粒子物理学标准模型的建立中发挥了核心作用——规范对称性决定了基本力的性质。

## 应用

诺特定理是理论物理学的基石：

1. **标准模型**：规范对称性 $SU(3) \times SU(2) \times U(1)$ 决定了强相互作用、弱相互作用和电磁相互作用的性质
2. **能量守恒**：时间平移对称性 → 能量守恒——热力学第一定律的深层原因
3. **动量守恒**：空间平移对称性 → 动量守恒——牛顿第三定律的推广
4. **角动量守恒**：旋转对称性 → 角动量守恒——开普勒第二定律的深层原因
5. **对称性破缺**：希格斯机制中的自发对称性破缺赋予粒子质量

## 与其他定理的关系

- **拉格朗日力学**：诺特定理建立在拉格朗日力学的框架之上
- **哈密顿力学**：守恒量对应于哈密顿量中的循环坐标
- **规范理论**：诺特定理是规范场论的理论基础
- **对称性与群论**：连续对称性由李群描述，诺特定理将李群的无穷小生成元与守恒量对应
- **熵增原理**：时间反演对称性的破缺与热力学第二定律

## 对称性的数学描述

诺特定理中的"对称性"由李群描述。常见的对称群包括：

- **平移群** $\mathbb{R}^n$：空间或时间的平移
- **旋转群** $SO(3)$：三维空间的旋转
- **洛伦兹群** $SO(3,1)$：时空的对称性（狭义相对论）
- **规范群** $U(1)$、$SU(2)$、$SU(3)$：标准模型中的规范对称性

每个李群有对应的李代数——由无穷小生成元张成。诺特定理将每个生成元与一个守恒量对应：

- 时间平移生成元 → 能量
- 空间平移生成元 → 动量
- 旋转生成元 → 角动量
- $U(1)$ 规范生成元 → 电荷

## 诺特定理的严格推导

设拉格朗日量 $L(q_i, \dot{q}_i, t)$ 在无穷小变换 $q_i \to q_i + \varepsilon \eta_i(q, t)$ 下不变，即 $\delta L = 0$。展开得：

$$\delta L = \sum_i \left(\frac{\partial L}{\partial q_i}\eta_i + \frac{\partial L}{\partial \dot{q}_i}\dot{\eta}_i\right) = 0$$

利用运动方程 $\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_i} = \frac{\partial L}{\partial q_i}$，上式化为：

$$\frac{d}{dt}\left(\sum_i \frac{\partial L}{\partial \dot{q}_i}\eta_i\right) = 0$$

因此守恒量为 $Q = \sum_i p_i \eta_i$，其中 $p_i = \partial L/\partial \dot{q}_i$ 是正则动量。对于时间平移 $\eta_i = \dot{q}_i$，守恒量 $Q = \sum_i p_i \dot{q}_i - L = E$ 正是能量。

## 离散对称性与守恒律

诺特定理只处理连续对称性——离散对称性（如空间反射 $x \to -x$ 或时间反演 $t \to -t$）不直接产生守恒律。然而，离散对称性在物理学中同样重要：

- **宇称守恒**（P）：物理定律在空间反射下不变——但在弱相互作用中被破坏（吴健雄实验，1957）
- **电荷共轭**（C）：粒子与反粒子的对称性
- **CP 对称**：C 和 P 的联合对称——在 $K^0$ 介子衰变中被发现部分破坏
- **CPT 定理**：任何洛伦兹不变的局域量子场论中 CPT 联合对称必须守恒

这些离散对称性的破缺揭示了宇宙的深层不对称性——例如 CP 破缺可能是宇宙中物质多于反物质的原因。

## 诺特定理在广义相对论中的微妙性

在广义相对论中，诺特定理的应用变得微妙。弯曲时空中没有全局的时间平移对称性，因此没有全局的能量守恒。诺特定理仍然适用，但守恒量变为协变守恒的应力-能量张量 $\nabla_\mu T^{\mu\nu} = 0$，这与平直时空中的 $\partial_\mu T^{\mu\nu} = 0$ 有本质区别。
