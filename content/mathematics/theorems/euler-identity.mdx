## 定理陈述

**欧拉恒等式**（Euler's Identity）是数学中最著名的等式：

$$e^{i\pi} + 1 = 0$$

这个等式将五个最重要的数学常数联系在一起：
- $e = 2.71828\ldots$——自然对数的底，分析学的核心
- $i = \sqrt{-1}$——虚数单位，复数的基础
- $\pi = 3.14159\ldots$——圆周率，几何学的基石
- $1$——乘法单位元
- $0$——加法单位元

同时涉及三种基本运算：加法、乘法、幂运算。

## 直觉理解

欧拉恒等式是**欧拉公式** $e^{ix} = \cos x + i\sin x$ 在 $x = \pi$ 时的特例。

理解欧拉公式的关键是：复指数 $e^{ix}$ 描述了复平面上单位圆上的点。当 $x = 0$ 时，$e^{i \cdot 0} = 1$（圆的最右端）。当 $x$ 增大时，点沿单位圆逆时针旋转。当 $x = \pi$ 时，点恰好旋转到圆的最左端，即 $e^{i\pi} = -1$。

所以 $e^{i\pi} + 1 = 0$ 说的是：从 1 出发，沿单位圆旋转半周（$\pi$ 弧度），到达 $-1$，再加 1 回到 0。这是复平面上的一个几何事实。

## 证明思路

欧拉恒等式由欧拉公式 $e^{ix} = \cos x + i\sin x$ 直接推出。

### 欧拉公式的证明

将 $e^{ix}$、$\cos x$、$\sin x$ 分别展开为麦克劳林级数：

$$e^{ix} = \sum_{n=0}^{\infty} \frac{(ix)^n}{n!} = 1 + ix - \frac{x^2}{2!} - i\frac{x^3}{3!} + \frac{x^4}{4!} + \cdots$$

分离实部和虚部：

$$e^{ix} = \left(1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots\right) + i\left(x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots\right)$$

$$= \cos x + i\sin x$$

代入 $x = \pi$：

$$e^{i\pi} = \cos\pi + i\sin\pi = -1 + 0i = -1$$

因此 $e^{i\pi} + 1 = 0$。

### 替代证明（微分方程方法）

考虑微分方程 $\frac{d}{dx}f(x) = if(x)$，$f(0) = 1$。

解为 $f(x) = e^{ix}$。同时 $g(x) = \cos x + i\sin x$ 也满足同一方程和初始条件。由解的唯一性，$e^{ix} = \cos x + i\sin x$。

## 历史背景

欧拉恒等式的历史与欧拉公式密不可分。

**罗杰·科茨**（Roger Cotes，1682—1716）在 1714 年发表于《自然科学会报》（Philosophical Transactions）的《Logometria》中，首次给出了等价形式 $ix = \ln(\cos x + i\sin x)$（指数化即得欧拉公式）。**欧拉**（Leonhard Euler，1707—1783）在 1748 年的《无穷分析引论》（*Introductio in analysin infinitorum*）中首次发表了 $e^{ix} = \cos x + i\sin x$ 的现代形式。

欧拉对复指数函数的理解建立在他对对数函数的复数推广之上。他注意到 $\ln(-1) = i\pi$（取主值），由此推出 $e^{i\pi} = -1$。

费曼在《费曼物理学讲义》（第一卷第 22 章）中称欧拉公式 $e^{ix} = \cos x + i\sin x$ 为"数学中最非凡的公式"（the most remarkable formula in mathematics）。

## 应用

欧拉恒等式及其推广（欧拉公式）在科学和工程中有广泛应用：

1. **交流电路分析**：用复指数表示正弦信号——$V(t) = V_0 e^{i\omega t}$ 简化了电路计算
2. **量子力学**：波函数 $\psi = Ae^{i(kx - \omega t)}$ 的核心是复指数
3. **信号处理**：傅里叶变换 $F(\omega) = \int f(t)e^{-i\omega t}dt$ 以欧拉公式为基础
4. **控制理论**：频域分析中传递函数的极点和零点在复平面上表示
5. **旋转表示**：计算机图形学中用 $e^{i\theta}$ 表示二维旋转

## 与其他定理的关系

- **欧拉公式**：$e^{ix} = \cos x + i\sin x$——欧拉恒等式是 $x = \pi$ 的特例
- **棣莫弗公式**：$(\cos\theta + i\sin\theta)^n = \cos n\theta + i\sin n\theta$——由欧拉公式推出
- **傅里叶分析**：欧拉公式将三角函数与复指数联系，是傅里叶分析的基础
- **微积分基本定理**：欧拉公式的证明依赖于幂级数展开，而幂级数理论建立在微积分之上
- **代数基本定理**：欧拉公式与复数的极坐标表示密切相关

## 为什么它被称为"最美丽的公式"

欧拉恒等式 $e^{i\pi} + 1 = 0$ 被广泛认为是数学中最美丽的公式，原因在于：

1. **简洁性**：用最简单的符号表达了最深刻的关系
2. **意外性**：五个看似无关的常数——$e$（分析学）、$i$（代数学）、$\pi$（几何学）、$1$（算术）、$0$（集合论）——被一个等式联系
3. **深刻性**：它不是偶然的巧合，而是反映了复指数函数、三角函数和圆的几何之间的深层联系
4. **统一性**：涉及加法、乘法和幂运算三种基本运算

费曼称之为"数学中最非凡的公式"。数学家哈代说："如果有什么东西能让数学家心动，那就是美。"

## 欧拉公式的其他特例

欧拉公式 $e^{ix} = \cos x + i\sin x$ 在不同 $x$ 值处给出不同的恒等式：

- $x = 0$：$e^{i \cdot 0} = 1$——平凡但正确
- $x = \pi/2$：$e^{i\pi/2} = i$——虚数单位是 $90°$ 旋转
- $x = \pi$：$e^{i\pi} = -1$——欧拉恒等式
- $x = 2\pi$：$e^{2i\pi} = 1$——$2\pi$ 旋转回到原点

每个特例都有深刻的几何意义：$e^{i\theta}$ 表示复平面上角度为 $\theta$ 的旋转。

## 复数的极坐标表示

欧拉公式给出了复数的极坐标表示：

$$z = r e^{i\theta} = r(\cos\theta + i\sin\theta)$$

其中 $r = |z|$ 是模，$\theta = \arg(z)$ 是辐角。这种表示使得复数的乘法和除法变得极其简单：

$$z_1 z_2 = r_1 r_2 e^{i(\theta_1 + \theta_2)}$$

即：模相乘，辐角相加——复数乘法就是旋转加缩放。

## 欧拉公式在物理学中的核心地位

欧拉公式 $e^{ix} = \cos x + i\sin x$ 是现代物理学不可或缺的工具：

**量子力学**：波函数 $\psi(\mathbf{r}, t) = A e^{i(\mathbf{k} \cdot \mathbf{r} - \omega t)}$ 是薛定谔方程的平面波解。复指数表示使得叠加原理的计算极为简洁——相位因子 $e^{i\phi}$ 编码了量子干涉的全部信息。量子力学的概率幅必须是复数——实数不足以描述干涉现象。

**电磁学**：时谐电磁场 $\mathbf{E}(\mathbf{r}, t) = \text{Re}[\mathbf{E}_0(\mathbf{r})e^{-i\omega t}]$ 用复指数表示正弦振荡。麦克斯韦方程在频域中变为代数方程——复指数表示将微分方程转化为代数方程。

**信号处理**：傅里叶变换 $F(\omega) = \int_{-\infty}^{\infty} f(t)e^{-i\omega t}dt$ 将信号分解为复指数分量。离散傅里叶变换（DFT）和快速傅里叶变换（FFT）是数字信号处理的核心算法——从 MP3 压缩到 MRI 成像，无处不在。

**交流电路**：阻抗 $Z = R + iX$ 将电阻和电抗统一为复数。欧姆定律 $V = ZI$ 在复数域中成立——电路分析从微分方程变为复数代数。

## $e$、$\pi$、$i$ 的深层联系

欧拉恒等式将三个看似无关的常数联系在一起，但它们之间存在深层的数学联系：

**$e$ 与 $\pi$ 的联系**：$e^{i\pi} = -1$ 表明 $e$ 和 $\pi$ 通过虚数单位 $i$ 联系。更直接的联系是高斯积分 $\int_{-\infty}^{\infty} e^{-x^2}dx = \sqrt{\pi}$——指数函数和圆周率在一个积分中相遇。

**$i$ 的几何意义**：$i$ 是 $90°$ 旋转——$i^2 = -1$ 表示两次 $90°$ 旋转等于 $180°$ 旋转。复数乘法 $z \mapsto iz$ 将复平面上的点逆时针旋转 $90°$。

**$e$ 的分析学意义**：$e$ 是唯一使得 $\frac{d}{dx}e^x = e^x$ 的底数——它是"自然增长"的数学表达。$e^{i\theta}$ 是"自然旋转"——虚指数的增长就是旋转。

这三个常数分别来自分析学（$e$）、几何学（$\pi$）和代数学（$i$），它们在欧拉公式中的统一反映了数学三大分支的深层联系。
