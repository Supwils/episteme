# 刚体转动与转动惯量

## 关键词

刚体; 转动惯量; 力矩; 角动量; 欧拉方程; 进动; 章动; 主轴; 平行轴定理; 垂直轴定理; 陀螺效应; 科里奥利力

---

## 第1页 · 破除误解

**标题：转动惯量不是质量的"旋转版本"——它依赖质量分布，是张量而非标量**

许多课本把转动惯量 $I = \sum_i m_i r_i^2$ 定义为"物体对转动的惯性"，并类比线运动中的质量，暗示二者只是"不同类型的惯性"而性质相似。但有一个根本区别：质量是标量（与方向无关），而**转动惯量是二阶对称张量**（inertia tensor）$\mathbf{I}$，其分量依赖于坐标轴的选择：$I_{ij} = \int (\delta_{ij}r^2 - r_i r_j)\rho\,dV$。对于任意刚体，总存在三个互相垂直的**主轴**（principal axes），使惯量张量对角化，主轴方向的转动惯量称为主转动惯量（$I_1, I_2, I_3$）。只有当转轴恰好沿主轴方向时，角动量才与角速度平行；一般情形下，角动量方向与角速度方向不同——这是陀螺进动等现象的根源。

另一个常见混淆是"刚体是理想化，实际不存在"。从工程角度，刚体是一个极好的近似（金属结构的弹性形变远小于尺寸），而且刚体的分析精确适用于工程中大量的旋转体——发动机曲轴、飞轮、卫星的姿态控制转轮，乃至地球本身（作为近似刚体的进动分析）。现代陀螺仪（MEMS惯性传感器、激光陀螺）是刚体转动力学的直接技术应用。

## 第2页 · 现场还原

**标题：欧拉方程与"乾坤大挪移"——刚体力学的数学建立**

刚体力学的系统数学处理由莱昂哈德·欧拉（Leonhard Euler）在18世纪中期建立。欧拉在1750至1765年间发展了刚体动力学的完整理论：引入了惯量张量的概念，建立了描述刚体在无力矩情形下转动的**欧拉方程**（Euler's equations，1758年），推导出了描述刚体定点转动（如陀螺）的欧拉角（Euler angles）参数化，还给出了刚体的动能公式 $T = \frac{1}{2}I_1\omega_1^2 + \frac{1}{2}I_2\omega_2^2 + \frac{1}{2}I_3\omega_3^2$（在主轴系中）。

欧拉方程在主轴系中的形式：

$$I_1\dot{\omega}_1 - (I_2 - I_3)\omega_2\omega_3 = N_1$$

$$I_2\dot{\omega}_2 - (I_3 - I_1)\omega_3\omega_1 = N_2$$

$$I_3\dot{\omega}_3 - (I_1 - I_2)\omega_1\omega_2 = N_3$$

其中 $\omega_1, \omega_2, \omega_3$ 是角速度在主轴系中的分量，$N_1, N_2, N_3$ 是力矩分量。对于无力矩情形（$N_i = 0$），方程变为非线性，但可以用椭球积分精确求解——结果是刚体的旋转轨迹（极迹，polhode）在惯量椭球面和能量椭球面的交线上运动。当 $I_1 = I_2 \neq I_3$（轴对称体，如子弹）时，出现规则的圆锥进动（Euler's disk motion，或称欧拉进动）。

帕雷德—达朗贝尔方程（d'Alembert's principle）和拉格朗日方法为刚体问题提供了另一套等价但往往更简便的分析工具。拉格朗日方法用欧拉角（$\phi, \theta, \psi$）描述刚体的三自由度定点转动，通过计算拉格朗日量并代入欧拉-拉格朗日方程，可以系统处理复杂约束条件下的刚体运动（如不打滑的滚动圆盘、陀螺在引力场中的运动）。

## 第3页 · 结构解释

**标题：平行轴定理、垂直轴定理与复合转动**

**平行轴定理**（Steiner's theorem）：物体对过质心的轴的转动惯量为 $I_c$，则对平行于该轴、相距 $d$ 处的轴的转动惯量为：$I = I_c + Md^2$（$M$ 为总质量）。这一定理将复杂形状对任意轴的转动惯量化为对质心轴的计算，是工程计算中最常用的刚体工具之一。

**垂直轴定理**（仅适用于薄平板）：若薄板在 $xy$ 平面内，则对垂直板面的轴（$z$ 轴）的转动惯量等于对两个互相垂直的面内轴的转动惯量之和：$I_z = I_x + I_y$。常用转动惯量公式：均匀细棒（长 $L$，过中点垂直）$I = ML^2/12$；均匀圆盘（半径 $R$，过中心垂直）$I = MR^2/2$；均匀实心球 $I = 2MR^2/5$；均匀球壳 $I = 2MR^2/3$。

**角动量定理**：力矩 $\boldsymbol{\tau} = d\mathbf{L}/dt$，$\mathbf{L} = \mathbf{I}\cdot\boldsymbol{\omega}$（张量运算）。对固定轴转动：$\tau = I\alpha$（类比 $F = ma$）。**能量定理**：$W = \int\tau\,d\theta = \Delta(\frac{1}{2}I\omega^2)$（转动动能）。**滚动物体**兼有平动和转动，总动能 $T = \frac{1}{2}Mv^2 + \frac{1}{2}I_c\omega^2$，其中 $v = R\omega$（滚动不打滑条件）。实心球（$I = 2MR^2/5$）滚到斜面底部时，总动能中5/7为平动动能，2/7为转动动能；空心球壳（$I = 2MR^2/3$）转动动能比例更高，故同一斜面上空心球比实心球慢——这一判断可用可乐罐、水瓶等简单实验验证。

## 第4页 · 代价与争议

**标题：陀螺进动、地球进动与卫星姿态控制**

**陀螺进动**（gyroscopic precession）是刚体力学最反直觉的现象之一：一个快速自旋的陀螺，其轴受到引力力矩（向下），但轴不是向下倒，而是**绕竖直轴缓慢旋转**（进动），进动角速度 $\Omega_p = N/(I_3\omega_s)$（$N$ 为力矩，$I_3$ 为对称轴的转动惯量，$\omega_s$ 为自旋角速度）。这一现象从牛顿定律的角度可以精确导出：力矩等于角动量的时间变化率，力矩使角动量矢量 $\mathbf{L}$ 的方向（而非大小）改变，导致 $\mathbf{L}$ 绕力矩方向进动。

地球本身就是一个巨大的陀螺：由于地球不是完美球形（赤道半径比极半径大约21 km），月球和太阳的引力产生力矩，使地球自转轴绕黄道法线进动（**地轴进动**，precession of equinoxes，岁差），周期约25,772年。这导致"岁差"（equinoctial precession）——分点（春分、秋分）沿黄道向西缓慢移动，每年约50.3角秒。古代天文学家（喜帕克斯，约公元前134年）已观测到岁差，这是人类最早的精密天文测量之一。此外，地球在28个月内还有一个约9米幅度的科达运动（Chandler wobble），源于地球的自由章动（非刚性地球的弹性，理论预测与实际观测的14个月差异至今有争议）。

**卫星姿态控制**是刚体力学在航天工程中的核心应用。卫星维持正确指向（对地/对太阳/对天体）需要精确控制三轴姿态。常用执行机构：反作用轮（reaction wheel，卫星本体内安装飞轮，改变飞轮转速产生反作用力矩）、控制力矩陀螺（CMG，control moment gyroscope，改变飞轮轴向产生力矩，力矩更大，用于空间站）、磁力矩器（磁棒与地球磁场相互作用，仅在低轨可用）。国际空间站（ISS）使用4个CMG提供主姿态控制，每个CMG含一个以6600 rpm旋转的100 kg飞轮，最大力矩约258 N·m。

## 第5页 · 长期遗产

**标题：从MEMS陀螺仪到量子旋量——转动惯量的技术与科学延伸**

**MEMS陀螺仪**（Micro-Electro-Mechanical System gyroscope）是智能手机、无人机、汽车电子稳定程序（ESP）中无处不在的微型传感器。MEMS陀螺仪利用**科里奥利效应**（Coriolis effect）而非经典宏观陀螺：一个振动的微机械结构（振动梁或音叉）在旋转时受到与旋转角速度和振动速度垂直的科里奥利力 $\mathbf{F}_{\rm Cor} = -2m\boldsymbol{\Omega}\times\mathbf{v}$，通过电容传感测量这一小力。MEMS陀螺仪尺寸约1—3 mm，角速度分辨率约 $0.01 °/\text{s}/\sqrt{\text{Hz}}$，是现代消费电子和航空航天导航的不可缺少的器件（每年产量约数十亿个）。

**激光陀螺**（Ring Laser Gyroscope，RLG）和**光纤陀螺**（Fiber Optic Gyroscope，FOG）是高精度惯性导航的核心传感器，利用**萨格纳克效应**（Sagnac effect）：在旋转参考系中，沿旋转方向和逆旋转方向传播的光（或其他波）走过相同路径需要不同时间，产生相位差 $\Delta\phi = 4\pi A\Omega/(\lambda c)$（$A$ 为封闭光路面积，$\Omega$ 为旋转角速度，$\lambda$ 为波长，$c$ 为光速）。激光陀螺的角速度分辨率可达约 $10^{-7}$ rad/s，用于军事飞机（F-22、F-35）、潜艇（无GPS环境的水下导航）和弹道导弹（惯性制导）。

**量子力学中的自旋**是一种无经典对应的内禀转动。电子自旋（$s = 1/2$，单位 $\hbar$）不能用经典旋转的刚体描述，但它与刚体角动量有数学上的类似性（对易关系相同，总角动量由轨道角动量和自旋耦合：$\mathbf{J} = \mathbf{L} + \mathbf{S}$）。原子核的自旋是核磁共振（NMR）和磁共振成像（MRI）的基础——原子核在外磁场中的进动频率（拉莫尔频率，$\omega_L = \gamma B_0$，$\gamma$ 为旋磁比，$B_0$ 为外磁场）正是NMR测量和MRI成像所利用的信号。从宏观陀螺到量子自旋，"转动"概念贯穿了物理学的所有尺度。

## 事实卡

- **卡1**：欧拉（1758年）建立刚体动力学完整理论，引入惯量张量、欧拉角和欧拉方程，是现代刚体力学的数学奠基。
- **卡2**：地球自转轴因引力力矩以约25,772年的周期进动（岁差），古希腊天文学家喜帕克斯约公元前134年已观测到这一现象。
- **卡3**：MEMS陀螺仪利用科里奥利效应，尺寸约1—3 mm，每年产量数十亿个，是智能手机、无人机和汽车稳定系统的核心传感器。
- **卡4**：NMR/MRI利用原子核在外磁场中的进动频率（拉莫尔频率）进行医学成像，是刚体进动在量子层面的类比应用。
