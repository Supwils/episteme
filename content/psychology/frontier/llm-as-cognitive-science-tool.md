# 大语言模型：认知科学的新工具与新研究对象

2023 年，一篇题为《大型语言模型通过了通用语言理解评估（GLUE）的所有子任务》的论文受到广泛关注——不是因为它的结论令人惊喜，而是因为它让认知科学家面临了一个奇怪的处境：我们设计的用来测量"人类般语言理解"的基准，现在被一个没有身体、没有童年、没有情感体验的统计机器轻松通过了。

这让认知科学家不得不问一个令人不舒服的问题：**我们真的知道"理解"是什么吗？**

大语言模型（Large Language Models，LLMs）正在以两种完全不同的方式改变认知科学：**作为工具**，它们提供了以前不可能实现的研究手段；**作为研究对象**，它们提供了一个非生物的"类认知系统"，让研究者能够以前所未有的方式检验认知理论。

## 破除误解：LLM 既不是"真的理解"，也不是"只是统计"

围绕 LLM 的认知科学讨论常常陷入两种极端：

一种极端声称 LLM 展现出了"真正的智能"或"理解"——GPT-4 在法学院考试中超过大多数人类考生，在医学资格考试中表现在前 10% 区间，在很多推理任务中超越了普通大学生。

另一种极端则认为 LLM "只是在做统计预测"，根本不可能真正推理——它们只是在复现训练数据中的模式，没有真正理解词语的意义，更没有与现实世界的接地（grounding）。

认知科学家的立场通常比这两种极端都更审慎：**LLM 的性能模式（哪些任务出奇地强、哪些出奇地弱）本身是一个值得严肃研究的现象**，它可以帮助我们更清楚地理解认知能力的结构——无论 LLM 是否"真正理解"。

## LLM 作为认知科学研究工具

### 1. 大规模心理学研究的新被试池

传统心理学研究受限于获取人类被试的成本和时间——每年 Mechanical Turk 等平台已经部分缓解了这一问题，但数量和多样性仍然有限。LLM 提供了一个新的可能：**用 LLM 模拟人类被试的反应，快速预筛实验设计，或在人类研究之前检验假设**。

Argyle 等人（2023）在《政治分析》发表研究，展示了 GPT-3 能够准确模拟特定人口统计学特征（年龄、教育、政治倾向）的受访者的调查回答分布，误差在可接受范围内[^argyle2023]。他们将此称为"硅基样本"（silicon sampling）。

**但这条路有重要边界**：LLM 的"回答"反映的是训练数据中的统计模式，而非真实人类的心理过程。将 LLM 模拟作为替代人类被试，在研究"态度"和"信念"时可能有一定价值，但在研究需要真实情绪体验、身体感知或行为的课题时，几乎不适用。

### 2. 预测神经科学实验结果

2024 年发表于《自然·人类行为》的研究发现，LLM（包括 GPT-4）在预测神经科学实验结果方面超越了人类专家[^luo2024]。研究者向模型提供实验设计摘要，要求预测结果，并与领域专家的预测进行比较。这个发现本身充满争议——它是 LLM 真正"理解"了神经科学，还是仅仅因为训练数据中包含了大量已发表的实验结果？

答案可能更平凡：LLM 对已发表文献的"记忆"使其在这类任务上具有天然优势，但这与"理解机制"无关。不过，这一工具在文献检索和假设生成阶段确实有实际价值。

### 3. 自然语言任务的精密实验平台

LLM 提供了在极为精密控制的条件下研究语言理解的新平台。认知语言学家可以系统性地测试：当某个词语的上下文细微变化时，LLM 的反应如何变化？这种操控性研究在人类被试上受到实验成本的严重限制，在 LLM 上可以以极低成本反复进行。

例如，Patel 等人（2022）用 LLM 测试了真实世界知识和语言理解的分离，发现某些 LLM 在语言形式的任务上表现优异，但在需要"知道世界实际运作方式"的任务上系统性失败——这为区分"语言能力"与"世界知识"提供了新的实验工具。

## LLM 作为认知科学的研究对象

这是一个更具原创性、也更具争议性的研究方向。

### 认知偏差：人类特有的还是统计的产物？

心理学家花费数十年记录了人类决策中的系统性偏差——锚定效应、框架效应、可得性启发等。一个令人着迷的发现是：**某些版本的 LLM 也表现出类似的"偏差"模式**。

Binz 和 Schulz（2023）发现，GPT-3 在一系列经典启发式偏差任务（Wason 选择任务、基础率忽视、锚定效应）上，表现出与人类非常相似的错误模式[^binz2023]。这引发了一个问题：这是因为 LLM 学习了人类的思维模式，还是因为这些"偏差"实际上是任何基于统计的信息处理系统在特定结构问题上的必然结果？

更有趣的是，当研究者使用 GPT-4 重复同样的实验时，部分偏差消失了——GPT-4 在某些推理任务上比 GPT-3 和普通大学生都**更接近规范理性**[^ullman2023]。这个"偏差随模型能力改善而消失"的模式，可能暗示这些偏差在 LLM 中是训练数据的统计模式，而不一定是"深层认知架构"的产物——这反而为我们理解**人类偏差**是否有更深的认知架构根源提供了对照。

### 类比推理：测量真实推理还是模式匹配？

Mitchell 和 Hofstadter 曾用类比推理（analogical reasoning）作为人类认知灵活性的核心指标。2022 年，Webb 等人发现 GPT-3 在经典类比推理任务（如"A 比 B 大，B 比 C 小，谁最大？"的变体）上超越了人类平均水平，且在新颖的类比结构上也表现良好[^webb2023]。

这个发现被乐观者解读为"LLM 实现了真正的类比推理"，被怀疑者解读为"LLM 可能在训练数据中见过这些任务的变体，只是在做模式匹配"。区分这两种解释需要设计"完全没有出现在任何训练数据中"的任务——这在实践中极为困难，因为互联网上几乎包含了无限多的推理任务变体。

### 语言接地问题（Grounding Problem）

LLM 的最深刻的认知科学挑战是**接地问题**（grounding problem）：词语的意义是否必须来自与现实世界的直接感知联结？

LLM 从未看见过一朵玫瑰，从未感受过热或痛，但它们能够流利地谈论玫瑰的颜色、火的温度。这是否意味着：一、意义可以仅从语言内部（符号之间的关系）获得，无需感知接地；二、LLM 处理的是符号关系而非意义，它们所做的不是"理解"而是"形式操作"？

这不是一个纯粹的技术问题，而是认知科学和语言哲学的核心问题。Winograd 模式（需要常识推断的代词消歧任务）和类似设计的基准测试，正是为了测量"仅靠符号关系是否足够"而设计的，但 LLM 已经在大多数此类基准上超过了人类——这使得"LLM 缺乏接地"的直觉判断与"LLM 在任务上表现良好"的实证发现之间产生了张力。

## LLM 与人类认知的系统性对比

以下是 2020 年代认知科学研究中较为一致的发现（尽管仍在快速更新中）：

| 认知维度   | 人类优势                              | LLM 特点                                         |
| ---------- | ------------------------------------- | ------------------------------------------------ |
| 推理效率   | 少量样本即可学习（few-shot learning） | 需要大量训练数据                                 |
| 迁移灵活性 | 跨域泛化能力强                        | 分布外泛化仍显著弱于人类                         |
| 常识推理   | 基于身体体验，稳健                    | 表面流利，深层常识有系统性缺口                   |
| 因果推理   | 可识别机制性因果                      | 仍以关联而非因果推理为主                         |
| 社会认知   | 高度发展，依赖情绪和意图读取          | 在"他心推断"任务上表现参差，对隐含社会线索不敏感 |
| 类比推理   | 强大的结构映射                        | 在形式化类比任务上可与人类媲美，但新颖性受限     |
| 知识更新   | 学习-遗忘动态灵活                     | 参数权重在训练后固定，不能实时更新               |

## 机器心理学的涌现：一个新分支

2023 年以来，"机器心理学"（machine psychology）逐渐被认知科学家和心理学家视为一个独立的研究分支。其方法论是：**用心理学的实验工具（认知测试、人格量表、道德判断任务、感知实验）系统性地评估 LLM，并将结果与人类数据进行对比**。

Binz 等人发起的 CogBench 项目（2023）是这个方向的代表性努力——他们开发了一套标准化的认知评估任务，可以用于跨模型比较，类似于人类认知测试的"标准化测验"[^cogbench]。

这个方向面临的核心方法学问题是：心理学测验是为人类设计的，其生态效度（ecological validity）对 LLM 是否适用？一个 LLM 在"情绪识别任务"上的"正确答案"，反映的是它训练数据中的统计模式，还是某种类似情绪感知的功能状态？这个问题目前没有共识答案。

## 代价与争议

**"随机鹦鹉"假说的持续张力**：Emily Bender 等人（2021）提出的"随机鹦鹉"（Stochastic Parrots）框架认为，LLM 只是在"形式上重组"训练数据中的语言模式，没有任何语义内容或意义[^stochastic]。这个框架的支持者认为，用心理学概念（理解、推理、偏差）来描述 LLM 是危险的误导。反驳者则认为，如果某个系统在所有可观察的输出上无法与"真正理解"区分，那"它只是鹦鹉"与"它真正理解"之间的区别是否还有实证意义？这触及了认知科学的哲学基础。

**基准污染**：许多认知测试任务在互联网上公开存在，LLM 可能在预训练阶段见过这些任务的变体，导致"良好表现"并不代表真正的能力，而是"记忆"[^gpt4benchmark]。研究者正在开发更系统的基准隔离方法。

**快速迭代带来的研究困难**：今天对某一 LLM 的认知研究结论，可能在新版本模型发布后即告失效。这使得认知科学领域的"积累性知识"建设变得困难——研究的对象不断更新，不同于人类认知的相对稳定性。

## 未知的边界

- **意识与体验**：LLM 是否有任何形式的"内在状态"或"主观体验"？这不是技术问题，而是哲学问题，但它与认知科学的核心议题深度相关。目前没有科学工具能够回答这个问题。
- **涌现能力的机制**：为什么某些认知能力（如多步推理）在模型规模超过某个门槛后"涌现"？这个涌现是真实的能力跳跃，还是测量方式的人工产物？
- **真实世界接地的必要性**：具身 AI（embodied AI，有传感器和执行器的系统）与纯文本 LLM 在认知能力上有何根本差异？这对理解人类认知为何需要身体具有深远意义。
- **LLM 作为认知理论的"猴子扳手"**：如果一个 LLM 能够以高精度预测人类在某个认知任务上的反应，这是否意味着当前的认知理论（及其提出的心理机制）是可以被抛弃的，因为"统计近似"已经足够？还是说，机制性解释（认知神经科学）和功能性解释（统计预测）服务于不同目的，两者并不可替代？

## 跨域连接

- **语言哲学（维特根斯坦）**：维特根斯坦的"意义即使用"理论（meaning is use）在 LLM 时代获得了新的测试场——如果一个系统的语言使用与人类无法区分，它是否也拥有语言的意义？这个讨论与[[语言哲学]]的核心问题直接相关。
- **图灵测试与心灵哲学**：图灵（1950）提出的"模仿游戏"（Imitation Game）作为智能标准，在 GPT-4 时代实际上已经被很多情境下通过——但这是否意味着"真正的智能"实现了？[[心灵哲学]]中的"中文房间"思想实验（Searle）获得了新的现实参照。
- **数学（Transformer 架构与注意力机制）**：LLM 的核心是 Transformer 架构，基于自注意力（self-attention）机制和矩阵运算。理解 LLM"能做什么、不能做什么"的理论边界，与线性代数和计算复杂性理论密不可分。
- **进化心理学**：人类认知能力是亿万年进化和几十年个体发展的产物，包含大量"进化压力"留下的结构（面部识别、语言习得的关键期、社会情绪）。LLM 仅仅通过文本预测训练，在几周内习得了部分类似能力——这为区分"进化不可缺少"与"可从数据中习得"的能力提供了前所未有的实验对照。
