# 春秋战国（前770年 - 前221年）

## 一、破除误解：不只是"百家争鸣"的思想花园

流行叙事将春秋战国浪漫化为"百家争鸣"的思想盛世，仿佛诸子在安静的书斋中从容论道。这一图景遮蔽了三个关键事实：第一，所谓"百家"诞生于持续五百余年的暴力兼并，思想竞争的底色是生死存亡的制度竞赛；第二，诸子之中多数是政治行动者——孔子周游列国推销政纲，商鞅以变法获车裂之刑，韩非入秦即遭毒杀——思想与权力从未分离；第三，"百家争鸣"的经典叙述建立在传世文献之上，而郭店楚简（1993年出土）、上博简等出土文献显示，早期儒道关系远比后世"儒道对立"的框架复杂，老子文本的年代与学派归属至今仍有重大争议。将这一时代简化为"思想自由的黄金年代"，既低估了暴力的规模，也高估了我们对先秦思想图景的认知完整性。

## 二、现场还原：铁器、城市与新型战争

### 政治格局

周幽王被犬戎所杀（前771年），周平王东迁洛邑，周天子权威急剧衰落。春秋初期尚有约一百四十个诸侯国，经不断兼并至战国仅余七雄——齐、楚、燕、韩、赵、魏、秦。春秋五霸（齐桓公、晋文公、楚庄王、秦穆公，及宋襄公或吴王阖闾/越王勾践，名单因文献而异）以"尊王攘夷"为旗号，实际争夺的是会盟主导权与势力范围。

### 铁器革命

春秋晚期至战国，冶铁技术从块炼铁发展到生铁铸造，铁制农具与牛耕的推广极大提高了农业生产力。这是中国历史上第一次真正的技术革命——更多的剩余粮食支撑了更大规模的人口、城市和常备军。战国时期铁制武器（剑、戟、箭簇）逐步取代青铜兵器，改变了战争形态。

### 城市化与商业

战国七雄的都城规模空前：齐国临淄人口可能达数十万，"临淄之途，车毂击，人肩摩"（《战国策》）。金属货币广泛流通，各国铸币形制各异（刀币、布币、圜钱），跨国贸易催生了大商人阶层——吕不韦以商人身份入秦为相，正是这一时代的缩影。

### 战争形态演变

春秋时期以车战为主，规模有限，贵族礼仪仍约束战争行为（宋襄公"不鼓不成列"即其遗风）。战国时期步兵与骑兵成为主力，战争烈度剧增：

- **桂陵之战**（前354年）：围魏救赵
- **马陵之战**（前342年）：孙膑击败庞涓
- **长平之战**（前260年）：白起歼灭赵军主力

> **长平之战人数争议**：传统文献（《史记》）称白起"坑杀赵卒四十万"。现代学者对此数字多有质疑：以战国时期赵国总人口推算，动员四十万青壮男性在后勤上几乎不可能；考古发掘的坑杀遗骸规模约为数万具。多数学者认为实际被杀战俘在十至二十万之间，"四十万"可能是包括后勤民夫在内的夸大数字，或为后世增饰。

## 三、结构解释：制度竞争与士人崛起

### 变法运动

战国的本质是制度竞赛——谁能最有效地动员人力、物力和军力，谁就能生存：

| 国家 | 改革者   | 核心内容                         | 效果                       |
| ---- | -------- | -------------------------------- | -------------------------- |
| 魏国 | 李悝     | 尽地力之教，制定《法经》         | 魏国率先崛起               |
| 楚国 | 吴起     | 削减贵族特权                     | 改革者被杀，成效有限       |
| 秦国 | 商鞅     | 废井田、奖军功、行县制、编户齐民 | 最彻底，使秦成为虎狼之师   |
| 韩国 | 申不害   | 术治（君主驭臣之术）             | 短期有效，长期不足         |
| 赵国 | 赵武灵王 | 胡服骑射，引入骑兵战术           | 军事提升，但未触动根本制度 |

商鞅变法的关键在于：以军功爵制取代世卿世禄，以编户齐民直接控制人口，以县制取代封邑——这是一套将社会彻底军事化的体制。

### 士人阶层的崛起

封建体制下"学在官府"，知识被贵族垄断。春秋末期，孔子首开私学，"有教无类"打破了教育的阶层壁垒。至战国，"士"成为一个活跃的游动阶层——他们不拥有土地和世袭权力，却凭借知识、辩才和谋略在各国之间流动，充当谋士、将领、外交官和思想家。苏秦佩六国相印、张仪以连横破合纵、商鞅以卫人入秦变法——"士无定主"是这个时代的鲜明特征。这一阶层的出现，是中国从封建贵族政治向官僚帝国转型的关键机制。

### 诸子百家

思想流派的分化回应着不同的政治方案：

- **儒家**：孔子（前551—前479年，仁、礼、中庸、有教无类）、孟子（前372—前289年，性善论、民贵君轻）、荀子（前313—前238年，性恶论、礼法并重）
- **道家**：老子（无为而治、道法自然）、庄子（逍遥游、齐物论）
- **法家**：韩非子（法、术、势结合）、商鞅（以法治国）、李斯
- **墨家**：墨子（兼爱、非攻、尚贤、尚同），组织严密，擅长守城技术
- **兵家**：孙武（《孙子兵法》）、孙膑（《孙膑兵法》）
- **名家**：公孙龙（白马非马）；**阴阳家**：邹衍（五行学说）；**纵横家**：苏秦、张仪

> **出土文献的修正**：1993年湖北荆门郭店楚墓出土竹简约八百余枚，包含早期儒家和道家文献，其中《老子》简本与传世本有重大差异，显示《道德经》的文本形成是一个漫长的过程，而非一人一时之作。上博简中的《孔子诗论》也修正了我们对早期儒学与《诗经》关系的理解。出土文献提醒我们：传世的"诸子百家"图景经过了汉代编纂者的筛选和重构。

## 四、代价与争议

### 战争的人口代价

整个春秋战国时期，战争持续约五百五十年。据《史记》等文献的不完全统计，战国时期战死人数可能达数百万。长平之战的坑杀只是最著名的一例。频繁的征兵、徭役和战乱对农业人口造成了巨大压力。

### 变法的代价

商鞅变法虽然使秦国强盛，但其严刑峻法（连坐制、轻罪重罚）和对社会自由的压制，为秦统一后的暴政埋下了伏笔。吴起在楚国变法被贵族射杀，显示改革者往往付出生命代价。

### 史学争论

- **"封建"与"郡县"之争**：春秋战国是否代表从封建制向郡县制的线性进步？部分学者（如许倬云）强调地区差异巨大，不可一概而论。
- **"百家"分类问题**：汉代刘歆《七略》的分类框架是否准确反映了先秦思想的实际面貌？出土文献显示许多文本难以归入任何一家。
- **历史书写偏差**：现存文献以中原视角为主，楚国、中山国等非中原政权的历史叙述严重不足。

## 五、长期遗产

### 秦统一的制度基础

商鞅变法建立的县制、编户齐民、军功爵制，成为秦汉帝国的制度骨架。秦统一六国（前230—前221年）的序列——韩、赵、魏、楚、燕、齐——不过是战国制度竞赛的最终裁决。

```
前230年 灭韩 → 前228年 灭赵 → 前225年 灭魏
前223年 灭楚 → 前222年 灭燕 → 前221年 灭齐 → 天下一统
```

### 思想遗产的长程影响

儒家经汉代"独尊儒术"成为帝制中国的官方意识形态，影响长达两千年。法家的制度设计（律令、官僚、户籍）构成了帝制的行政骨架。道家则始终作为儒家的补充和矫正存在。这三家的互动，定义了中国传统政治文化的基本格局。

### 与其他文明的比较

春秋战国与古希腊城邦时代（前8—前4世纪）、印度列国时代（前6—前4世纪）大体同期，被雅斯贝尔斯称为"轴心时代"——多个文明独立产生了系统性的哲学和宗教思想。这一同步性至今没有令人满意的解释。

## 事实卡

1. **长平之战**（前260年）：秦将白起歼灭赵军主力，传统文献称坑杀四十万，现代学者估计实际战俘被杀十至二十万。此战奠定了秦统一的基础。
2. **商鞅变法**（前356年起）：废世卿世禄、行军功爵制、编户齐民、推行县制，使秦国从边陲弱国变为统一六国的虎狼之师。
3. **郭店楚简**（1993年出土）：约八百余枚竹简，包含早期儒道文献，修正了对先秦思想图景的认知，显示《道德经》文本经历了漫长的形成过程。
4. **铁器推广**（约前6—前4世纪）：冶铁技术从块炼铁发展到生铁铸造，铁制农具和武器的普及极大提高了生产力并改变了战争形态。
