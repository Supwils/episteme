## 数据架构

### 条目 ID 规范

所有知识条目使用统一的 ID 格式：`{时期}-{类型}-{名称}`，例如：

- `classical-event-fall-of-rome` — 古典时期事件：罗马帝国衰亡
- `medieval-figure-genghis-khan` — 中世纪人物：成吉思汗
- `modern-civilization-industrial-britain` — 现代文明：工业革命时期的英国

### 内容文件结构

```
knowledge-base/
├── 远古时期/概述.md          ← 时期总览
├── 远古时期/农业革命.md      ← 核心事件详情

```
