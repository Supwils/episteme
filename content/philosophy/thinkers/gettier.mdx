# 埃德蒙·盖梯尔：三页纸颠覆两千年

## 破除误解

盖梯尔（1927-2021）几乎是哲学史上最奇特的"单篇作者"——他凭一篇**仅三页**的论文永久改变了认识论，此后几乎再未发表学术文章。这常被当作轶事，却恰恰说明哲学进展未必依赖鸿篇巨制：一个**精确的反例**，足以推翻一个延续两千多年的定义。

另一个误解是以为盖梯尔提出了某种新理论；实际上他什么正面理论都没提，他只是**提出了问题**——而这个问题至今没有公认的答案。整个认识论领域在过去六十年里所做的大量工作，有相当部分是在回应这个三页纸提出的挑战。

关于盖梯尔本人：据他的同事描述，他极为低调，不喜欢学术圈的争论气氛，这可能是他几乎停止发表的原因之一。他1927年10月31日出生于巴尔的摩，2021年3月23日去世，活到93岁，亲眼看到了自己那篇文章引发的整整一代认识论的风云变幻。

## 生平脉络

- **1927年10月31日** 出生于美国马里兰州巴尔的摩
- 在康奈尔大学求学，受维特根斯坦和牛津日常语言哲学传统影响
- 任教于底特律的韦恩州立大学（Wayne State University）
- **1963年** 发表《被证成的真信念是知识吗？》（Is Justified True Belief Knowledge?），刊于《分析》（Analysis）杂志，仅约三页
- 据传他本不愿发表，只因职称晋升需要发表成果，才勉强投稿——这个故事的真实性有些疑问，但在哲学界广为流传
- 此后在马萨诸塞大学阿默斯特分校（University of Massachusetts Amherst）任职至退休
- 晚年几乎不再接受采访或参与学术活动
- **2021年3月23日** 逝世，享年93岁

## 传统的知识定义（JTB）

理解盖梯尔反例，需要先了解他颠覆的是什么。

自**柏拉图《泰阿泰德篇》**以来，知识被广泛地分析为"**被证成的真信念**（Justified True Belief，缩写 JTB）"。按照这个定义：

**S知道P，当且仅当：**
1. **P为真**（真理条件）
2. **S相信P**（信念条件）
3. **S的信念是被证成的**（有充分理由的）（证成条件）

两千多年里，这一分析虽有补充和修正，却从未被根本动摇。20世纪初，罗素、摩尔和阿耶尔在分析哲学框架内对这一定义进行了精确化，使其成为认识论的标准出发点。

**证成（justification）的含义**：并不要求证成必须是逻辑上的确证，只要求有"充分理由"相信。我相信外面正在下雨，因为我听到了雨声——这个听觉依据构成了我信念的"证成"。

## 盖梯尔反例

1963年的论文提供了两个反例。以下是其中一个的结构，以及后来哲学家们提出的更清晰版本：

**盖梯尔的原始案例（简化版）**：史密斯有充分理由相信"将得到工作的人口袋里有十枚硬币"——他看到琼斯口袋里有十枚硬币，且有强烈证据认为琼斯会得到这份工作。由此他推出了命题P："将得到工作的人口袋里有十枚硬币"。

结果：史密斯自己得到了工作，而**恰好他口袋里也有十枚硬币**——这他事先并不知道。

分析：
- P为真（将得到工作的史密斯口袋里确实有十枚硬币）✓
- 史密斯相信P ✓
- 史密斯有充分理由（证成）相信P（基于对琼斯的观察）✓

但我们的直觉强烈地认为：史密斯**并不知道**P，他只是**碰巧蒙对**了。他的证成（关于琼斯）与P为真的实际事实（关于史密斯自己）之间的联系是**运气性的**，而非知识所要求的那种恰当联系。

**更直观的版本（后人重构）**：你看表，时间显示2:00，你因此相信"现在是2:00"（你有充分理由：这块表一向准确）。但这块表恰好昨晚停了，而现在真的是2:00。你的信念为真、你相信它、你有证成——但你知道现在是2:00吗？直觉上，不知道——你只是碰巧对了。

## 盖梯尔问题与各派回应

盖梯尔反例的核心教训：**知识排斥某种认知运气**——被证成的真信念若只是"侥幸为真"，就不算知识。这便是"盖梯尔问题"：如何修补或重构知识的定义？

半个多世纪里，认识论者提出了大量方案，下面按思路分类梳理：

### 方案一：增加第四个条件

最直接的反应是给JTB加上第四个条件。

- **"无虚假引理"（no false lemma）**：S的证成不经过任何虚假的中间命题。在史密斯的案例中，他的推理隐含了"琼斯会得到工作"这一虚假信念。这个方案排除了许多盖梯尔案例，但也有反例：可以构造不经过虚假中间命题的盖梯尔型案例。

### 方案二：可靠主义（Reliabilism）

阿尔文·戈德曼（Alvin Goldman）在1967年提出：真正的知识要求信念来自**可靠的认知过程**（reliable cognitive process）。"可靠"是指该过程在通常情况下产生真信念。看清楚的眼睛、正常工作的记忆、良好的推理——这些是可靠的。碰巧正确的猜测不是。

**优势**：可以排除许多盖梯尔型案例，因为"看停着的表以为是2:00"不是一个在通常情况下可靠的过程（表可能随时停）。

**挑战**：如何确定"可靠"的范围？同一个认知过程在某些环境下是可靠的，在其他环境下不是（视觉在正常光线下可靠，在极度昏暗中不可靠）。这引发了"广义可靠主义"的环境问题。

### 方案三：诺齐克的真理追踪（Truth-Tracking）

罗伯特·诺齐克（Robert Nozick）在1981年提出：S知道P，当且仅当S的信念"追踪"真理——即满足以下反事实条件：
1. 若P为假，S就不会相信P（敏感性条件）
2. 若P为真，S就会相信P（粘附性条件）

对"停着的表"案例：若现在不是2:00，你看表仍会相信是2:00（因为表停了）——不满足敏感性条件，所以不算知识。

**优势**：优雅地处理了大多数盖梯尔案例。

**挑战**：诺齐克的条件在封闭原则（closure principle）上遇到麻烦。如果我知道P，且P蕴含Q，直觉上我也应该知道Q。但诺齐克的条件在某些情况下违反这一原则。

### 方案四：德性认识论（Virtue Epistemology）

恩内斯特·索萨（Ernest Sosa）和琳达·扎格泽布斯基（Linda Zagzebski）等人提出：知识不应该被分析为信念加上外部条件，而应该理解为认知能力（intellectual virtue/faculty）的**成功运用**。

知识是一种**成就（achievement）**——它不仅要求信念为真，而且要求这个真来自于认知者自身能力的成功运用，而非运气。

**类比**：一位弓箭手射中了靶心，但如果是因为突然刮来的风把射偏的箭吹回了靶心，这不算技艺的展示。真正的成就要求成功与能力之间的恰当联系。

**挑战**：如何精确界定"能力的成功运用"？认知能力在不同情境下的表现本身可能受运气影响。

### 方案五："知识优先"（Knowledge First）

蒂莫西·威廉姆森（Timothy Williamson）在《知识及其限度》（2000年）中提出了一个根本性的转向：不是将知识分析为更基本的概念（真信念+条件），而是将知识本身视为**认识论的基础原始概念**，用知识来分析其他概念（如证成、证据）。

证成不是知识的条件，恰恰相反：知识才是证成的来源——你有理由相信P，当且仅当你知道P（或知道某些使P可信的东西）。

**优势**：从根本上规避了盖梯尔问题——因为JTB分析本身被放弃了。

**挑战**：威廉姆森的框架在某些哲学家看来过于保守，缺乏对知识内部结构的解释。

## 哲学争议：盖梯尔反例真的有效吗？

尽管大多数哲学家接受盖梯尔反例，也有少数反驳：

**挑战一：直觉的文化相对性**。近年来的"实验认识论"（experimental epistemology）研究表明，对盖梯尔案例的直觉反应存在跨文化差异。不同文化背景的被试对"史密斯知道P吗"的回答不完全一致。这引发了哲学方法论问题：如果我们的哲学分析依赖直觉，而直觉本身是文化相对的，那分析的结论有多大的普遍性？

**挑战二："知道"是否有多种意思**？有些哲学家（语境主义者，如斯图亚特·科恩，Stuart Cohen）认为"知道"在日常语言中是语境敏感的——在低标准语境中，史密斯"知道"P；在高标准语境中（如法庭），他不"知道"。如果是这样，盖梯尔问题不过是在不同语境中切换，而非揭示了知识定义的根本缺陷。

**挑战三：证成的错误来源**。一些哲学家认为，盖梯尔案例的关键不是"运气"，而是证成本身来自"错误的事实"——史密斯的证成基于对琼斯的观察，但真正使P为真的事实是关于史密斯自己的。这建议修改证成条件而非整个JTB框架。

## 影响与遗产

一篇三页的论文催生了认识论一个完整的子领域。盖梯尔问题不仅是技术难题，更触及"知识、真理与运气"之关系的根本：我们看重知识，正因它是一种**不靠运气**地把握真理的成就。

**在哲学方法论上的意义**：盖梯尔论文也是分析哲学"反例方法"（counterexample method）的典范——哲学分析的有效性取决于它能否经受反例的检验，而一个精确的反例就足以推翻一个分析。这一方法论塑造了分析认识论此后几十年的研究模式。

**在认识论之外**：盖梯尔案例揭示的"认知运气"问题，在法律认识论（证据的可信度）、人工智能（机器学习模型是否"知道"某事，还是只是在数据集上碰巧正确）等领域同样具有实践意义。

**实验认识论**：盖梯尔问题的争议催生了"实验哲学"（experimental philosophy）——用调查研究来检测哲学家依赖的直觉是否普遍，以及是否受语境效应影响。这一领域在21世纪成为哲学研究的新兴方向。

## 后续讨论：什么是知识的核心？

盖梯尔问题至今没有公认的解决方案，但它推动了哲学家澄清一个更深的问题：**知识为什么对我们重要**？

苏格拉底在《美诺篇》中已经注意到：被证成的真信念（他说"正确的观点"）与知识（他说"通过绑住而固定的知识"）在实用上可能没有差别——一个猜对了路的人和一个真正知道路的人，带领你到达终点的效果是一样的。那么，知识相对于"正确的猜测"究竟多了什么？

盖梯尔告诉我们："被证成"还不够。但他没有告诉我们还需要什么。这个问题仍然开放，仍然值得探索。

## 跨领域连接

- **法学**：盖梯尔案例与证据法——什么算"知道"证人说谎，还是只是碰巧怀疑？
- **人工智能**：深度学习模型的"知识"——模型在测试集上正确是否意味着它"知道"，还是只是运气？
- **心理学**：实验认识论——用调查研究检测直觉的文化相对性和语境敏感性
- **科学哲学**：证成与真理的关系——科学理论的"证成"是否能带来真知？
