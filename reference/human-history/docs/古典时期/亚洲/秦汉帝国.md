# 秦汉帝国（前221年 - 220年）

## 一、破除误解：不只是"伟大的统一"

流行叙事将秦汉帝国简化为"统一天下、奠定中华"的英雄史诗。这一框架遮蔽了若干关键事实：第一，秦的统一依赖商鞅变法建立的极端军事动员体制，其代价是全社会的严刑峻法和大规模徭役，秦朝存续仅十五年即崩溃——统一本身并非天然正当或可持续；第二，"焚书坑儒"的受害者主要是方士（为秦始皇求仙药的术士），而非纯粹的儒家学者，《史记》原文为"诸生四百六十余人皆坑之咸阳"，其中方士占多数，后世将其简化为"迫害儒生"是叙事重构；第三，汉朝并非简单地"继承"秦制，而是经历了从黄老无为到儒法合流的漫长制度实验，其间还有王莽新朝这一重大断裂。将秦汉视为铁板一块的"帝国奠基"，模糊了制度演进的真实曲折。

## 二、现场还原：帝国的建立与运转

### 秦朝（前221年 - 前206年）

秦始皇统一六国后推行一系列标准化措施：
- **统一文字**：小篆为标准文字，隶书逐渐通行
- **统一度量衡与货币**：半两钱为标准货币
- **修驰道**：以咸阳为中心的全国道路网络
- **行郡县制**：废分封，全国设三十六郡（后增至四十余郡）
- **连接长城**：将秦、赵、燕三国旧长城连为一体

重大工程包括万里长城、秦始皇陵（兵马俑1974年出土）、灵渠（连接长江与珠江水系）。

### 楚汉之争与西汉建立

秦朝在陈胜吴广起义（前209年）后迅速崩溃。经过四年楚汉战争，刘邦击败项羽，前202年建立汉朝，定都长安。

### 汉武帝时期（前141年 - 前87年在位）

汉武帝刘彻在位五十四年，是汉朝国力的巅峰：
- **独尊儒术**：采纳董仲舒建议，设太学，以儒学为官方意识形态
- **北击匈奴**：卫青、霍去病多次出击匈奴，收复河套、河西走廊
- **开通西域**：张骞两次出使西域（前138年、前119年），打通丝绸之路
- **盐铁官营**：国家垄断盐铁等重要经济资源
- **推恩令**：巧妙削弱诸侯王势力

### 汉匈关系：不只是"击败"

汉匈关系远比"卫青霍去病大败匈奴"复杂。汉初国力不足，高祖刘邦亲征匈奴被困白登山（前200年），此后实行"和亲"政策——以宗室女嫁匈奴单于、岁赠丝帛酒食换取边境和平，持续约六十年。武帝时期转为军事进攻，但长期战争耗竭国力，晚年下"轮台罪己诏"反思。此后汉匈关系交替出现和亲、战争与朝贡，并非单向的"征服"。

### 王莽新朝（8年 - 23年）

西汉末年，外戚王莽篡位建立新朝，推行激进改革：将土地收归国有、禁止奴婢买卖、改革币制。这些"托古改制"的理想主义政策引发社会动荡，绿林赤眉起义推翻新朝。王莽新朝是中国历史上罕见的"儒学乌托邦实验"，其失败深刻影响了后世对制度改革的态度。

### 东汉（25年 - 220年）

光武帝刘秀复兴汉朝，定都洛阳。东汉时期的成就包括：
- 班超经营西域（73年起），重建对西域诸国的控制
- 蔡伦改进造纸术（105年）
- 张衡发明地动仪（132年）
- 佛教正式传入中国（明帝时期白马寺）

东汉后期外戚与宦官交替专权，最终导致黄巾之乱（184年）和军阀割据，220年曹丕篡汉，三国时代开始。

## 三、结构解释：帝国的制度骨架

### 法律体系

秦汉建立了中国最早的系统化法律体系。睡虎地秦简（1975年出土）显示秦律涵盖田律、仓律、工律、效律等，条文精密，执行严格。汉承秦律，但萧何作《九章律》有所简化。汉代引儒入法，"春秋决狱"（以儒家经义断案）开创了法律儒家化的进程。

### 丝绸之路：双向文化交流

丝绸之路不仅是贸易通道，更是文明双向交流的管道：

```
长安 → 河西走廊 → 敦煌
    → 北道：车师 → 大宛 → 康居 → 安息 → 大秦（罗马帝国）
    → 南道：鄯善 → 于阗 → 大月氏 → 安息
```

- **中国输出**：丝绸、漆器、铁器、造纸术
- **西方输入**：葡萄、石榴、胡桃、苜蓿、汗血马、玻璃器、金银币
- **双向传播**：佛教自印度经中亚传入中国；中国的冶铁技术向西传播

罗马帝国对中国丝绸的需求极大，老普林尼曾抱怨罗马贵妇的丝绸购买导致黄金外流。丝绸在罗马被称为"赛里斯"（Seres，意为"丝国"）之物。

### 科技与文化

| 成就 | 人物/时间 | 意义 |
|------|----------|------|
| 造纸术 | 蔡伦改进（105年） | 革命性书写材料，促进知识传播 |
| 地动仪 | 张衡（132年） | 世界最早的地震方位检测装置 |
| 《史记》 | 司马迁（约前91年完成） | 中国纪传体史学的开创之作 |

## 四、代价与争议

### 秦制的代价

秦始皇的统一伴随着巨大的人力代价：修长城征发数十万民夫，建秦始皇陵和阿房宫用工七十余万，严刑峻法使"赭衣塞路，囹圄成市"。秦朝人口约两千万，常年征发的徭役劳动力达数百万。这种极端压榨是秦朝十五年即亡的根本原因。

### 儒学独尊的代价

汉武帝"罢黜百家，独尊儒术"虽未完全禁止其他学派，但确立了儒学的官方垄断地位。这在提供统一意识形态的同时，也限制了思想的多样性。法家、墨家、名家等学派逐渐边缘化。

### 史学争论

- **秦始皇评价**：是"千古一帝"还是"暴君"？现代史学倾向于承认其制度创新的同时，强调统一过程的暴力本质。
- **和亲政策评价**：是屈辱妥协还是务实外交？近年学界更倾向于肯定和亲在维持边境和平方面的实际效果。
- **汉武帝的功过**：开疆拓土的同时耗竭国力，"有亡秦之失而免亡秦之祸"（《资治通鉴》），幸有轮台罪己诏及时调整。

## 五、长期遗产

### 帝制模板

秦汉确立的中央集权郡县制、官僚选拔制度、统一文字和度量衡，成为此后两千年中国帝制的基本模板。"汉族"、"汉字"、"汉语"的名称均源于汉朝。

### 儒法合流

汉代形成的"外儒内法"治理模式——以儒学为意识形态合法化工具，以法家律令为实际治理手段——成为中国帝制时代政治文化的核心特征。

### 丝绸之路网络

汉代开通的丝绸之路，将东亚、中亚、西亚和地中海世界连接为一个松散但持续的交流网络。佛教的传入是这一网络最深远的文化后果之一，深刻改变了中国的精神世界。

### 与罗马帝国的比较

秦汉与罗马大致同期（前3世纪—3世纪），两者都建立了跨区域的大帝国，但制度路径截然不同：秦汉依赖郡县制和官僚直接控制，罗马则更多依赖公民权扩展和地方自治。这一差异深刻影响了此后东西方的政治传统。

## 事实卡

1. **睡虎地秦简**（1975年湖北云梦出土）：一千余枚竹简，包含大量秦律条文，是研究秦代法律制度的第一手资料，显示秦律之精密远超此前想象。
2. **张骞出使西域**（前138年、前119年）：第一次出使被匈奴扣留十三年，第二次成功联络大月氏等国。两次出使开启了中原与中亚的正式外交和贸易往来。
3. **王莽新朝**（8—23年）：中国历史上最具理想主义色彩的改制实验，以"托古"为名推行土地国有、币制改革等激进政策，因脱离实际而迅速失败。
4. **和亲政策**：汉初（约前198—前135年）以宗室女嫁匈奴单于并岁赠财物换取边境和平，是中国历史上最早的系统性外交妥协政策。
