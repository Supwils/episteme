# 人类历史知识图谱 - 项目规划

## 项目愿景

构建一个从远古文明到当代的交互式人类历史知识图谱网站，让用户能够以可视化、沉浸式的方式探索人类发展历程。

## 技术栈决策

> 以 `开发规范.md` 和 `website/AGENTS.md` 为权威来源。本节仅做摘要。

### 构建工具
- **Vite 8+** - 现代前端构建工具，极速HMR

### 语言
- **JavaScript (ES Modules, ES2022)** - 纯原生JS，不使用TypeScript

### 动画与可视化
- **GSAP (GreenSock) 3.12+** - 专业级动画库，用于时间线滚动动画、手写效果、路径动画
- **SVG** - 手绘风格的图标、地图和插图
- **Canvas** - Atlas知识图谱与关系图的底层渲染

### 样式
- **原生CSS + CSS Custom Properties** - 设计令牌主题系统
- **不使用Tailwind CSS** - 保持原生方案

### 路由
- **Browser History Router（自研）** - 轻量级History API路由

### 包管理
- **npm**

### 禁止引入
React、Vue、Angular、TypeScript、Tailwind CSS、D3.js。本项目坚持原生方案。

## 网站功能设计

### 1. 主时间线
- 垂直滚动的时间线，从公元前10000年到2025年
- GSAP ScrollTrigger驱动的滚动动画
- 各时期节点可展开详情
- 手绘风格的SVG标记

### 2. 知识图谱
- Canvas力导向图（原生实现）
- 人物、事件、文明之间的关系连线
- 可缩放、拖拽、筛选
- 节点点击跳转到详情页

### 3. 大洲视图
- SVG手绘世界地图
- 点击大洲查看该区域历史
- 丝绸之路、大航海路线的路径动画

### 4. 人物传记
- 手写风格的传记卡片
- 生平时间线
- 关联人物网络

### 5. 交互式学习路径
- 推荐学习路线
- 知识点之间的前置关系
- 学习进度追踪

## 页面路由

> 实际路由以 `website/AGENTS.md` 为准。

```
/                     # 首页（Hero + 统计 + 时代卡片 + 迷你时间线 + 人物）
/timeline             # 完整时间线视图（筛选 + 分组 + 模态详情）
/atlas                # Atlas知识图谱（Canvas分层探索）
/graph                # 关系图（Canvas力导向图）
/map                  # 大洲地图视图（SVG地图 + 区域详情）
/figures              # 人物页（筛选 + 卡片网格）
```

## 视觉风格

- **手绘质感**：SVG手写线条、纸张纹理背景
- **古卷轴风格**：展开式动画、卷轴边框
- **暖色调**：羊皮纸色、深褐色、金色点缀
- **中文书法**：标题使用书法字体
- **粒子效果**：历史节点用粒子连线
