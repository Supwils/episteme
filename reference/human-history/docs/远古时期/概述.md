# 远古时期（约公元前10000年 - 前3000年）

## 概述

远古时期是人类文明的黎明时代。在这数千年间，人类从游牧狩猎的生活方式逐步过渡到定居农耕，这一转变被称为"农业革命"或"新石器革命"，是人类历史上最深刻的变革之一。

## 关键发展

### 农业的诞生
约在公元前10000年，人类开始在肥沃的新月地带（Fertile Crescent）驯化植物和动物。小麦、大麦、绵羊、山羊相继被驯化。几乎同时，中国的黄河和长江流域也开始种植粟和水稻。

### 定居与村落
农业带来了定居生活。人类开始建造永久性住宅，形成村落。杰里科（约公元前9000年）和加泰土丘（约公元前7500年）是最早的城镇之一。

### 陶器与技术
- 陶器的发明（约公元前10000年）
- 纺织技术的出现
- 石器磨制技术的完善
- 最早的金属冶炼（铜，约公元前5000年，安纳托利亚地区）

### 信仰与艺术
- 岩画艺术（拉斯科洞窟、阿尔塔米拉）
- 女性雕像（维纳斯雕像）
- 原始宗教与萨满信仰
- 丧葬仪式的出现

## 历史启示

> 农业革命使人类从自然的被动适应者变为主动改造者，这一转变奠定了此后所有文明的基础。但同时也带来了社会分层、私有制和战争——进步从来都是双刃剑。
