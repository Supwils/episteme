# 亚里士多德（前384年 - 前322年）

## 破除误解：亚里士多德不是"只靠冥想的书斋哲学家"

通俗叙事常将亚里士多德描绘为一位坐而论道的纯粹思想家。这一形象忽略了他生命中最关键的政治脉络：他生于马其顿崛起的时代，父亲尼各马可是马其顿国王阿明塔斯三世的御医，他本人则受腓力二世之邀担任王子亚历山大的教师（前343—前336年）。亚里士多德的哲学体系——从政治学到生物学——都与帝国扩张的资源、网络与视野密不可分。他不是远离权力的人，而是权力结构的参与者与受益者。

## 时代背景：马其顿崛起与帝国视野

亚里士多德生活在希腊城邦独立时代的黄昏。前338年喀罗尼亚战役，马其顿腓力二世击败雅典—底比斯联军，终结了城邦作为独立政治体的历史。其子亚历山大（前356—前323年）东征波斯、埃及与中亚，建立起从地中海到印度河的空前帝国。这一时代有两个特征塑造了亚里士多德的思想：第一，城邦的衰落使"最好的政体是什么"成为紧迫的理论问题；第二，帝国扩张带来的地理与生物知识前所未有的丰富——亚历山大东征期间随军学者采集的标本与记录，为亚里士多德的生物学研究提供了珍贵素材。他笔下对500余种动物的描述，许多来自帝国视野所及的远方。

## 生平

### 早年与柏拉图学院
- 生于马其顿的斯塔基拉（Stageira），父亲是马其顿国王的御医
- 17岁前往雅典，进入柏拉图学院学习
- 师从柏拉图20年，柏拉图称他为"学院的灵魂"
- 但逐渐发展出与老师不同的哲学观点："吾爱吾师，吾更爱真理"

### 游历时期（前347年 - 前335年）
- 柏拉图去世后离开雅典，在小亚细亚和莱斯博斯岛研究生物学
- 受腓力二世邀请前往马其顿宫廷，担任亚历山大的教师（前343年）
- 这段经历使他接触帝国行政实践与广域地理知识

### 创办吕克昂（前335年 - 前323年）
回到雅典，创办自己的学校——吕克昂学园：
- 边散步边讲学，学生被称为"逍遥学派"
- 大量研究和写作，覆盖逻辑、物理、生物、伦理、政治、诗学等领域
- 据传亚历山大资助其研究，提供大量动植物标本

### 晚年
亚历山大大帝去世（前323年）后，雅典反马其顿情绪高涨。亚里士多德以"不给雅典第二次犯哲学之罪"（指苏格拉底之死）为由逃离雅典，次年在优卑亚岛去世。

## 核心贡献

### 逻辑学
发明了形式逻辑（三段论），西方逻辑学两千年的基础：
```
大前提：所有人都会死
小前提：苏格拉底是人
结论：苏格拉底会死
```

### 形而上学与自然哲学
- **实体论**：个别事物为第一实体，共相为第二实体
- **四因说**：质料因、形式因、动力因、目的因——以目的论解释自然变化
- **物理学**：虽然地心说等观点后被推翻，但他开创了系统观察与分类方法

### 伦理学与政治学
- **美德伦理**：幸福（eudaimonia）是灵魂合乎美德的活动，中庸之道是美德的核心
- **政治学**："人是政治的动物"——城邦生活是人之为人的条件
- 对政体的分类（君主制、贵族制、共和制及其变体）影响深远

### 生物学
- 对500多种动物进行分类与描述，部分解剖研究精确度惊人
- 提出"自然阶梯"概念，将生物从低到高排列
- 达尔文称其为"古代最伟大的博物学家"

## 争议：被忽视的阴暗面

### 为奴隶制辩护
亚里士多德在《政治学》中明确论证"有些人天生是奴隶"，认为奴隶是"有灵魂的工具"。这一观点在古典世界并非独特，但它被后世用作合理化奴隶制的理论资源。19世纪美国南方蓄奴者大量引用亚里士多德为奴隶制辩护。

### 对女性的看法
他认为女性是"不完整的男性"，理性能力低于男性，不适合参与政治。这一观点同样被中世纪与近代学者广泛援引，压制女性权利运动。

### 目的论阻碍科学发展
亚里士多德的四因说——尤其是"目的因"——在中世纪被经院哲学奉为权威，成为挑战新科学的阻力。伽利略、牛顿等人必须先突破亚里士多德物理学的框架，才能建立近代力学。弗朗西斯·培根批评亚里士多德"以逻辑代替观察"，虽有不公，但指出了问题核心。

## 长期遗产：从巴格达到巴黎

### 阿拉伯学者的保存与传播
亚里士多德的著作在西罗马帝国灭亡后几乎失传于欧洲，但在伊斯兰世界被完整保存、翻译与注释。叙利亚学者早在6世纪便开始翻译其作品；阿拔斯王朝巴格达"智慧宫"将大量希腊文著作译为阿拉伯文。

- **阿维森纳**（伊本·西那，980—1037）：融合亚里士多德哲学与伊斯兰神学，其《治疗论》是中世纪最重要的哲学百科全书
- **阿威罗伊**（伊本·鲁世德，1126—1198）：为亚里士多德撰写详尽注释，主张理性与信仰不矛盾，其思想经犹太哲学家迈蒙尼德传播至拉丁欧洲
- 亚里士多德哲学经由阿拉伯—拉丁翻译运动（12—13世纪），重新进入欧洲知识界

### 托马斯·阿奎那的综合
13世纪，托马斯·阿奎那（1225—1274）将亚里士多德哲学与基督教神学系统融合，证明信仰与理性可以并行。这一"经院哲学综合"成为天主教会官方哲学，影响至今。但丁称亚里士多德为"智者之师"。

### 对犹太哲学的影响
迈蒙尼德（1138—1204）在《迷途指津》中以亚里士多德哲学框架重新阐释犹太教义，成为中世纪犹太哲学的里程碑。

### 近代科学的"反面参照"
讽刺的是，亚里士多德最大的近代影响恰恰是作为"需要被推翻的权威"。伽利略的落体实验、牛顿的惯性定律，都是对亚里士多德物理学的直接反驳。但这种反驳本身证明了他的持久影响力——直到17世纪，欧洲大学课程仍以亚里士多德为核心。

## 事实卡

- 亚里士多德师从柏拉图20年，后发展出以经验观察为基础的哲学体系，与柏拉图理念论分道扬镳
- 他担任亚历山大大帝的教师（前343年），其生物学研究受益于帝国扩张带来的标本与地理知识
- 他的著作在中世纪经阿拉伯学者阿维森纳、阿威罗伊保存与注释，后经拉丁翻译运动重返欧洲
- 托马斯·阿奎那将亚里士多德哲学与基督教神学融合，塑造了天主教会的官方哲学传统
