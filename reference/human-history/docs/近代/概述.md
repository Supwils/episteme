# 近代（公元1500年 - 1800年）

## 概述

近代是世界历史的转折点。大航海时代将各个大陆连接在一起，科学革命颠覆了人类对自然的认知，启蒙运动重新定义了政治和社会。与此同时，全球权力格局经历深刻重组：欧洲大西洋沿岸国家借助海洋扩张获得了前所未有的跨洲影响力，但这一过程并非单向的"欧洲崛起"——它同时伴随着美洲文明的毁灭、非洲的奴隶贸易掠夺、以及亚洲帝国的持续竞争与内部转型。理解近代，必须同时阅读扩张者与被扩张者的故事。

## 关键主题

### 大航海时代
欧洲人寻找通往亚洲的新航路，意外"遭遇"了美洲（"发现"一词隐含欧洲中心视角，因为美洲早有数千万原住民居住）：
- 哥伦布到达美洲（1492年）
- 达·伽马到达印度（1498年）
- 麦哲伦—埃尔卡诺船队环球航行（1519年 - 1522年）

大航海的真正引擎是白银与全球贸易网络的连接。波托西（今玻利维亚）等美洲银矿在强制劳役（米塔制）下开采的白银，经塞维利亚流入欧洲，更经马尼拉大帆船横跨太平洋流向中国——明清中国对白银的巨大需求是这一全球货币循环的核心动力。与此同时，旧大陆病菌的传入造成美洲原住民人口在征服后的数十年间锐减（一种被广泛引用但仍有争议的估计高达九成），大西洋奴隶贸易则将上千万非洲人强制运往美洲种植园。理解大航海，必须同时看到欧洲的扩张、美洲的人口崩溃、非洲的掠夺与亚洲帝国（明清中国、莫卧儿、奥斯曼）依旧举足轻重的位置。

### 科学革命
```
哥白尼（1543年）：日心说
    ↓
伽利略（1609年）：望远镜观测
    ↓
开普勒（1619年）：行星运动定律
    ↓
牛顿（1687年）：万有引力定律
```

### 启蒙运动
理性、自由、平等的思想在欧洲传播：
- 洛克：自然权利、社会契约
- 伏尔泰：宗教宽容、言论自由
- 卢梭：人民主权、社会契约论
- 孟德斯鸠：三权分立

### 殖民扩张
欧洲列强在全球建立殖民帝国：
- 西班牙：中南美洲
- 葡萄牙：巴西、非洲沿海、亚洲
- 英国：北美、印度
- 法国：北美、西非
- 荷兰：东南亚

## 各大洲概况

### 欧洲
- 文艺复兴
- 宗教改革
- 科学革命
- 启蒙运动
- 重商主义

### 亚洲
- 中国：明清时期，从繁荣到有限开放与内向化（明初郑和下西洋后海禁政策、清初开海与限关交替，而非简单的"闭关锁国"）
- 印度：莫卧儿帝国从阿克巴的宗教宽容到奥朗则布的扩张与帝国收缩
- 日本：战国时代、德川幕府与锁国体制下的内部文化繁荣
- 奥斯曼帝国：从扩张到内部改革与挑战（16世纪苏莱曼大帝鼎盛后的行政调整、军事改革尝试与18世纪的变革压力）

### 美洲
- 美洲原住民文明被摧毁
- 殖民地的建立
- 大规模跨大西洋奴隶贸易

### 非洲
- 大西洋奴隶贸易对沿海与内陆社会的深远破坏
- 欧洲沿海据点
- 内陆王国的延续（如阿散蒂王国、刚果王国）

## 时间线

| 时间 | 事件 |
|------|------|
| 1453年 | 君士坦丁堡陷落——中世纪结束的标志性事件，为近代欧亚格局变化奠定背景 |
| 1492年 | 哥伦布到达美洲 |
| 1498年 | 达·伽马到达印度 |
| 1517年 | 马丁·路德宗教改革 |
| 1519年 | 麦哲伦船队开始环球航行 |
| 1543年 | 哥白尼发表《天体运行论》 |
| 1588年 | 英国击败西班牙无敌舰队 |
| 1600年 | 英国东印度公司成立 |
| 1607年 | 英国在北美建立第一个殖民地 |
| 1642年 | 英国内战 |
| 1687年 | 牛顿发表《自然哲学的数学原理》 |
| 1776年 | 美国独立宣言 |
| 1789年 | 法国大革命 |

注：1453年君士坦丁堡陷落严格来说属于中世纪晚期，但在本时间线中作为近代开始（1500年）的背景节点收录，因为它深刻影响了此后欧亚大陆的政治与贸易格局。

## 历史启示

> 近代是人类最矛盾的时代。科学革命和启蒙运动带来了理性和自由，但殖民扩张和奴隶贸易却造成了巨大的苦难。进步与野蛮并存，文明与压迫同在。理解这段历史，才能理解当今世界的不平等格局。
