# 当代（公元1945年 - 至今）

## 概述

当代是人类历史上变化速度最快的时代。从冷战的核恐惧到互联网的普及，从殖民体系的瓦解到全球化的深入，从登月到人工智能，人类在短短几十年间经历了前所未有的变革。

理解这一时代，需要避免"欧美单中心"的视角。1945年后真正塑造世界格局的，不仅是美苏的两极对峙，还有第三世界的集体登场：1955年万隆会议上29个亚非国家寻求第三条道路，到1960年"非洲年"一年内17个国家独立，全非洲最终诞生54个独立国家；不结盟运动（1961年贝尔格莱德首次峰会）一度涵盖全球过半人口。与此同时，中国从1949年世界上最贫穷的国家之一（据估计当时人均GDP不足百美元、文盲率超过80%）走向2010年世界第二大经济体，印度成为人口第一大国与软件服务中心，拉美在威权与民主之间反复探索。当代史是一部多中心的全球史。

## 关键主题

### 冷战（1947年 - 1991年）
美苏两个超级大国的对峙：
```
美国（资本主义阵营）          苏联（社会主义阵营）
    │                              │
北约（1949年）               华约（1955年）
    │                              │
    ├── 太空竞赛                    ├── 人造卫星（1957年）
    ├── 核武竞赛                    ├── 加加林太空飞行（1961年）
    ├── 古巴导弹危机（1962年）      ├── 阿富汗战争（1979年）
    └── 阿波罗登月（1969年）        └── 解体（1991年）
```

### 殖民体系瓦解与第三世界的兴起
```
1947年 - 印度、巴基斯坦从英国独立（伴随分治悲剧）
1949年 - 中华人民共和国成立
1954年 - 越南奠边府战役，终结法国在印度支那的殖民统治
1955年 - 万隆会议，29个亚非国家共商"第三条道路"
1957年 - 加纳独立（撒哈拉以南非洲第一个独立国家）
1960年 - "非洲年"，17个非洲国家独立
1961年 - 不结盟运动首次峰会（贝尔格莱德）
1990年 - 纳米比亚独立（非洲最后一个殖民地）
1994年 - 南非废除种族隔离，曼德拉当选总统
1997年 - 香港回归中国
1999年 - 澳门回归中国
```
去殖民化不是欧洲"恩准"的结果，而是亚非拉人民长期斗争的产物。全非洲最终形成54个独立国家；万隆精神与不结盟运动为它们提供了在冷战二元框架之外的话语空间。

### 信息时代
```
1947年 - 晶体管发明
1958年 - 集成电路
1969年 - 互联网前身ARPANET
1971年 - 微处理器
1976年 - 苹果公司成立
1983年 - 互联网（TCP/IP）
1989年 - 万维网
1998年 - 谷歌成立
2004年 - Facebook
2007年 - iPhone
2010年 - Instagram
2022年 - ChatGPT发布，引爆生成式AI革命
```
据国际电信联盟（ITU）估计，全球互联网用户已从1995年的数千万增长到2024年的约58亿、2025年的约60亿（约占世界人口四分之三）——但"数字鸿沟"依然存在：高收入与低收入国家在网络接入与速率上差距悬殊。

### 全球化
经济、文化和政治的全球互联：
- 世界贸易组织（WTO，1995年成立，中国2001年加入）
- 欧盟（1993年《马斯特里赫特条约》生效）
- 全球供应链与跨国生产网络
- 文化全球化与本土化的张力
- 反全球化运动与2010年代的"逆全球化"思潮

全球化既缩小了国家之间的差距（中国、印度等新兴经济体追赶），也在许多国家内部扩大了贫富差距——经济学家米拉诺维奇用"大象曲线"刻画了这一双重效应：全球中产（多在亚洲）和顶端1%受益最多，发达国家中下层相对停滞。

### 中国的崛起
```
1949年 - 中华人民共和国成立（当时人均GDP据估计不足百美元）
1958年 - 大跃进（1958—1962年大饥荒造成巨大人口损失）
1966年 - 文化大革命（1966—1976年）
1978年 - 改革开放
1997年 - 香港回归
2001年 - 加入WTO
2008年 - 北京奥运会、应对全球金融危机
2010年 - 超越日本成为世界第二大经济体
2020年 - 新冠疫情
2022年 - 中国空间站（天宫）完成在轨建造
```
改革开放四十余年间，中国经济年均增长近10%，使数亿人脱贫——据世界银行估计，中国脱贫人口占同期全球脱贫总数的相当大比重。这是人类历史上规模最大的减贫进程；但增长也伴随环境压力、贫富差距与区域不平衡等代价。

## 重大挑战

### 21世纪的全球性问题
- 气候变化：全球长期升温已达约1.3°C；据世界气象组织，2024年成为有记录以来最暖年份、首个全年均温超过1.5°C的年份（这是单一年度，并不等于《巴黎协定》长期目标已失败）
- 人口老龄化：东亚、欧洲生育率持续走低，多国进入深度老龄化社会
- 贫富差距：据估计全球最富有的1%人口掌握的财富份额显著高于底层一半人口
- 地缘政治冲突：从中东到俄乌战争（2022年起），大国博弈重新升温
- AI伦理：生成式AI引发就业、版权、虚假信息与"对齐"难题
- 公共卫生危机：新冠疫情暴露全球卫生治理的脆弱

数据说明：上述统计多为机构估算，会随口径与年份变化，应以"约/据估计"理解。

## 历史启示

> 当代史最大的教训是：技术是一把双刃剑。核能可以发电也可以毁灭城市，互联网可以连接世界也可以散布虚假信息，AI可以辅助人类也可以取代人类。人类能否驾驭自己创造的力量，是21世纪最大的考验。
