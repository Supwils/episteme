# 全站内容审计报告

**日期**：2026-06-09
**执行**：Claude Code（量化盘点 + 全量渲染爬取 + 8 领域并行子代理审查）
**范围**：8 个知识领域，content/ 下约 1060 个 mdx/md + ts 数据文件，约 558 万字符。

## 0. 执行摘要

平台内容的**散文质量普遍高于同类大众知识平台**：核心条目事实密度大、主动破除常见误解、引用规范、对前沿争议（哈勃张力、复制危机、DESI 暗能量）多能呈现为未决问题。真正的问题**不是写得不好，而是三类系统性缺陷**：

1. **数据层与内容层脱节** → 搜索 404、内容不可被发现（5 个领域共 ~180 个死链接 slug）。
2. **结构性重复与孤儿内容** → 同一主题多份维护、近一半天文内容无路由不可见、文件内整段重复。
3. **少量"幻觉级"事实硬伤** → 在世学者被写成已故、虚构合作者人名、算术错误、伪托名言。

**密度失衡的真相**：宇宙学(261K)、物理(325K)、生命科学(376K)字符量垫底，但物理"薄"主要是因为 **~165K 字符内容是孤儿**（见 §3）。先修管道，再写新内容，投入产出比最高。

## 2. 跨领域系统性问题（按优先级）

### P0 — 已在本次会话修复

- **`{#anchor}` 锚点字面渲染**：`MarkdownRenderer.tsx` 与 `thinker-detail/content-parser.ts` 此前不剥离 `## 标题 {#id}` 的锚点后缀，导致 20 个哲学页（schools/isms）+ thinkers 页正文与目录显示原始 `{#core-definition}` 文本。**已修复**（新增 `parseHeading()`，锚点作为显式 id），并经浏览器验证可见文本干净。
- **h4 不渲染**：thinker 解析器不支持 `#### `，整行降级为段落文本。**已修复**。

### P1 — 孤儿内容与缺失路由（物理/宇宙学，~165K 字符不可见）

### P2 — frontmatter schema 碎片化（全域）

几乎每个领域都有 2–3 套并存的 frontmatter 约定（字段名、era 枚举值、status 有无、.md/.mdx 混用）。除生命科学 dialogues 的 12 篇经 Zod 校验外，绝大多数内容未经 schema 校验，是上述结构问题的根因。

### P3 — 渲染器缺陷（数学专属，45 处）

`MathMarkdownRenderer.tsx`（数学不走 MDX，走自研解析器）：

- **粗体内公式不渲染**：`**$n=4$**` 直接输出 `$n=4$` 字面（33 处，18 文件）。
- **表格单元格内含 `|` 的公式被撕裂**：`$\max|\lambda_i|$`、`$|G|$` 等按 `|` 切列断裂（12 行，9 文件）。
- **建议**：改渲染器（`renderInline` 对 `**…**` 递归解析；表格切分避让 `$…$`）比改 45 处内容划算。

## 3. 各领域事实性错误（高置信度，节选）

完整清单见各子代理报告，此处列每领域最严重者：

- **心理学**：`theorists/otto-kernberg.mdx` 把在世学者（1928–）写成"2024 年去世，享年96岁"（正文重复 4 次）；`experiments/darley-latane-1968.mdx` 旁观者效应数据与站内另一文自相矛盾。
- **历史**：哥伦布页伊莎贝拉卒年 1505（应 1504）；二战事实卡"死亡"与正文"伤亡"3500万自相矛盾；events.js 邓巴数张冠李戴到黑猩猩；多处伪托名言（牛顿"计算疯狂"）。
- **经济学**：`great-depression.mdx` 日元贬值写成单位/方向皆错（实为升值）；`daniel-kahneman.mdx` 虚构合作者"克莱尔·格里科"；凯恩斯"班柯"误挂国际清算银行（应国际清算同盟）。
- **数学**：`group.mdx` 群论计数 96/4 算成 21（应 24）；`two-envelope.mdx` 悖论条件 x=100 写错（应 x>100）；`banach-tarski.mdx` 编造参考书作者；欧拉恒等式排版错 `a^{ei}+1=0`。
- **生命科学**：`octopus.mdx` "每条触手 2.4 亿个吸盘"（同文 L127 正确写 240 个）；珍·古道尔研究所成立年 1994（应 1977）；古道尔已于 2025-10 去世需补卒年。
- **哲学**：`what-is-consciousness.mdx`/`can-machines-think.mdx` 给在世的塞尔（1932–）虚构卒年 2022；维特根斯坦《逻辑哲学论》"约75个命题"（应 7 主命题/500余条）；取消主义 vs 物理主义代表标签颠倒。
- **天体物理**：巨引源距离 200 Mpc（应 65–80 Mpc，单位混淆）；DESI"DR1/2025年3月"（应 DR2/三年数据）；μ子 g-2 叙事滞后（2025 后张力已消解，恰是科学自我修正的好素材）。

**伪托/无出处引文是跨领域系统性风险**：历史、经济、数学、哲学、物理均有把流传语录/生成式编造的"名言"当真直接引用的情况。建议统一标注一手出处，无法溯源者改转述或标"传"。
