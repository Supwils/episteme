# docs/archive — 历史归档

本目录存放**已耗尽使命的一次性时间点报告**。它们记录的是某个具体日期的快照，其发现要么已被修复，要么已被更新的审计/工作日志取代。保留它们用于追溯，但它们**不是**真相源，新代理无需阅读。

真相源仍在 `docs/` 根目录（`任务清单.md`、`工作日志.md`、`工程原则.md`、`知识精神.md`、`聚焦方向.md` 等）。

| ------------------------ | ------------------------------------------------------------------- |
| `诊断报告-2026-06-07.md` | 2026-06-07 的一次性诊断快照，问题已修复或被后续审计取代             |
| `接管审计-2026-06-08.md` | 2026-06-08 的一次性接管审计，已被 2026-06-16 的五子代理全局审计取代 |
