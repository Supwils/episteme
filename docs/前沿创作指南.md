# 研究前沿创作指南（Frontier Authoring Guide）

> 本指南规定「研究前沿」板块所有文章的格式、写作标准与事实纪律。
> 黄金标准范文：`content/universe-physics/frontier/quantum-error-correction.md`。动笔前先读它。

---

## 1. 这个板块是什么

「研究前沿」回答一个问题：**当下（2020 年代），人类在这个领域正在推进的最前沿是什么？**

每篇文章聚焦一条**具体的、正在进行的前沿**，而不是一个已盖棺定论的教科书话题。它要讲清楚：

- 这条前沿想解决的**开放问题**是什么（为什么悬而未决）；
- **谁**在攻关（团队 / 机构 / 实验，真实存在）；
- 近年（约 2020–2026）有哪些**具体突破**，带**年份与来源**；
- 哪些仍然**未知**——诚实地标出边界，不假装已有答案；
- 与平台其他领域的**跨域连接**。

与已有板块的区别：知识库讲"已确立的知识"，前沿讲"**正在发生的知识**"——有悬念、有争议、有可能被推翻。

## 2. 文件位置与格式

- 路径：`content/<领域>/frontier/<英文-kebab-slug>.md`（**.md**，不是 .mdx）。
- slug 用英文小写连字符，例如 `protein-design-revolution.md`。
- 用 `gray-matter` frontmatter，正文用 Markdown（由站点的 `MarkdownRenderer` 渲染，支持 KaTeX `$...$` / `$$...$$`、表格、脚注 `[^id]`、引用块）。

### Frontmatter 必填 / 可选字段

```yaml
---
title: 蛋白质设计的革命 # 中文标题（必填）
title_en: The Protein Design Revolution # 英文标题（可选，建议填）
status: published # published / draft / stub（必填，正式内容用 published）
updated: 2026-06-12 # YYYY-MM-DD（必填）
category: 结构生物学 # 用于分组与配色（必填，同领域内尽量复用少数几个 category）
horizon: 2020s # 时间跨度标签，如 "2020s" / "近10年"（可选）
order: 2 # 同 category 内排序，数字小的在前（可选）
tags: # 3-6 个（可选但建议）
  - 蛋白质折叠
  - AlphaFold
researchers: # 代表性研究者/团队，真实存在（可选但建议）
  - David Baker（华盛顿大学）
  - DeepMind
institutions: # 机构/实验/计划（可选）
  - Baker Lab / Institute for Protein Design
related: # 同领域其他 frontier slug 或站内 slug（可选）
  - synthetic-biology
---
```

### 正文结构（建议骨架，可按主题调整）

1. **开篇钩子**：用一个具体的近年事件/数字切入（哪一年、谁、做成了什么）。
2. `## 破除误解`：先指出关于这条前沿的常见但过浅的理解。
3. `## 现场 / 核心问题`：把开放问题讲清楚，为什么难。
4. `## 谁在做、做到了哪一步`：团队、路线对比（**鼓励用表格**对比不同技术路线/学派）。
5. `## 代价与争议`：局限、失败路径、学界分歧、被高估之处。
6. `## 未知的边界`：诚实列出仍然没有答案的问题。
7. `## 跨域连接`：连到数学/物理/哲学/历史等其他领域（可用 `[[概念]]` 标注）。
8. `## 延伸阅读`：**真实**的论文/书目（期刊名、年份、arXiv/DOI 优先）。

深度要求：**正文非空行 ≥ 60**（旗舰深度长文，原始行数通常 150+）。段落不要写成长墙——一段聚焦一个判断，便于阅读。

## 3. 事实纪律（最高优先级，违反即返工）

本平台最大的信誉风险是"幻觉级"事实硬伤。前沿内容尤其危险，因为它涉及最新、最易编造的信息。

- **凡涉及近年（2020–2026）的具体进展，必须用 WebSearch 核实**：年份、机构、论文、数字、人名。
- **绝不编造**：不杜撰研究者姓名、论文标题、机构、数据。**不确定就写得概括**，宁可只描述方向，也不要编造一个看起来很具体的"事实"。
- **区分**"已确证的结果"与"理论预期 / 尚有争议的主张"，用语要分明。
- **在世 / 已故要核实**：不给在世学者编卒年。
- **引文要可溯源**：无法溯源的"名言"改为转述或标注"传"。
- **数字带单位、带来源**：误差范围、测量方法尽量交代。
- 写完后自检：文中每一个具体的年份/数字/人名，我能否指向一个真实来源？不能的，改写。

> 记住知识精神：宁可承认"目前学界尚无定论"，也不要伪装成定论。前沿的魅力恰恰在于它**尚未确定**。

## 4. 语言与风格

- 正文中文（专有名词保留英文：AlphaFold、CRISPR、JWST、arXiv 等）。
- "顶级教授讲课"式：有现场感、有结构、有争议意识、有跨域连接。
- 拒绝空洞形容词（"伟大的""震撼的"）和廉价励志结语——让事实本身说话。
- 代码/变量/slug 用英文。
