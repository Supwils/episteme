# 项目总览 — Universe Knowledge 知识平台

> 最后更新：2026-06-06

## 平台愿景

> 让任何人随时都能以美好的方式接触人类最重要的知识。

**Universe Knowledge** 是一个知识即服务平台（Knowledge as a Service），通过浏览器以可视化、沉浸式的方式探索多个核心知识领域：宇宙物理、人类历史、哲学思想、生命科学、经济学、心理学。

---

## 当前架构：统一应用

**所有代码运行在一个 Next.js 应用中**，通过 `localhost:3000` 统一访问。

```
universe-knowledge/
├── apps/
│   └── portal/                    ← 唯一活跃应用（Next.js 15）
│       ├── app/
│       │   ├── page.tsx           ← Portal 首页
│       │   ├── universe-physics/  ← 宇宙物理路由（28 条）
│       │   ├── human-history/     ← 人类历史路由（10 条）
│       │   └── philosophy/        ← 哲学思想路由（12 条）
│       ├── src-physics/           ← 物理前端代码（组件/场景/逻辑）
│       ├── src-history/           ← 历史前端代码（组件/渲染器/逻辑）
│       └── src-philosophy/        ← 哲学前端代码（组件/lib/逻辑）
│
├── content/                       ← 所有知识内容（与代码解耦）
│   ├── universe-physics/          ← 23 TS + 15 纹理 + 3 lib
│   ├── human-history/             ← 558 条目 + 104 篇知识库
│   ├── philosophy/                ← 229 MDX + 7 lib
│   └── life-science/              ← 84 物种 + 35 篇知识库 + 10 对话
│
├── packages/
│   ├── content/                   ← @universe/content 跨应用链接
│   └── ui/                        ← @universe/ui 共享组件（空壳）
│
└── docs/                          ← 平台文档
```

### 访问方式

```bash
pnpm dev    # 单一命令，单一端口

localhost:3000/                    → Portal 首页
localhost:3000/universe-physics/*  → 宇宙物理（28 路由）
localhost:3000/human-history/*     → 人类历史（10 路由）
localhost:3000/philosophy/*        → 哲学思想（12 路由）
```

### 代码与内容解耦

| 层级     | 目录            | 职责                          |
| -------- | --------------- | ----------------------------- |
| **内容** | `/content/`     | 所有知识文章、数据、纹理、MDX |
| **代码** | `/apps/portal/` | 组件、路由、逻辑、样式        |
| **工具** | `/packages/`    | 跨应用共享工具                |

## 应用全景

### 1. 宇宙物理（/universe-physics）

**核心体验**：跨尺度漫游。从可见宇宙（10²⁶ 米）逐层下钻到地球（10⁶ 米），每一层都有 WebGL 3D 场景和科学叙事。

**内容**：8 个宇宙尺度层 + 9 个物理板块 + 792 dataCards + 13 张行星纹理

**技术**：Next.js 15 + React Three Fiber + Three.js + Zustand + GSAP

**特性**：

- 三种场景过场（zoom / dissolve / tunnel）
- 5 步性能降级（Bloom → DPR → 粒子 → 后处理 → Toast）
- LOD 粒子系统 + 相邻层预加载
- 全局键盘导航 + ARIA 无障碍
- Web Vitals 性能监控

### 2. 人类历史（/human-history）

**核心体验**：从远古到当代的交互式历史图谱。时间线、Atlas 知识图谱、SVG 世界地图、人物传记。

**内容**：219 事件、235 人物、104 篇知识库文章

**技术**：Next.js 15 + GSAP + Canvas + CSS Custom Properties

**特性**：

- Ctrl+K 全局搜索
- 明暗主题切换
- 页面过渡动画
- 知识库浏览（/human-history/knowledge）

### 3. 哲学思想（/philosophy）

**核心体验**：文字驱动的哲学知识图谱。哲学家传记、流派脉络、思想实验、大问题。

**内容**：68 哲学家、38 流派、28 主义、21 思想实验、14 大问题、40 概念、20 对话 = 229 篇 MDX

**技术**：Next.js 15 + MDX + Framer Motion + Tailwind v4

**特性**：

- 跨内容类型交叉引用
- 时代筛选 + 搜索
- 108 个时间线事件
- 跨领域互文链接

### 4. 生命科学与进化（/life-science）

**核心体验**：从第一个有机分子到人类意识，探索40亿年生命演化的壮阔历程。

**内容**：8 个时代、84 物种、50 事件、23 科学家、6 大灭绝、35 篇知识库文章、10 篇对话

**技术**：Next.js 15 + Framer Motion + Tailwind v4

### 5. 门户（/）

**核心体验**：平台主入口。六大领域卡片、动画 Hero、全局搜索入口。

**技术**：Next.js 15 + Tailwind v4 + Server Components

### 6. 宇宙学（/cosmology）

**核心体验**：宇宙学专题内容。

### 7. 数学（/mathematics）

**核心体验**：数学专题内容。

### 8. 经济学（/economics）

**核心体验**：从古典经济学到行为经济学，探索经济思想的演进与现实应用。

**内容**：29 经济学家、12 理论、10 概念、6 案例研究、9 学派、4 辩论、4 对话

**技术**：Next.js 15 + MDX + Framer Motion + Tailwind v4

**特性**：

- 经济模拟实验（供需模拟、囚徒困境、拍卖模拟等）
- 经济学家按时代分类（古典/新古典/现代/当代）
- 跨内容类型交叉引用

### 9. 心理学（/psychology）

**核心体验**：从弗洛伊德到认知偏差，探索人类心智的奥秘。

**内容**：36 理论家、26 实验、16 现象、6 学派、6 心理障碍、4 辩论、4 对话

**技术**：Next.js 15 + MDX + Framer Motion + Tailwind v4

**特性**：

- 经典心理学实验重现与解读
- 认知偏差与心理现象可视化
- 心理学家按时代分类（经典/现代/当代）

### 10. 知识图谱（/knowledge-graph）

**核心体验**：跨领域交互式知识图谱可视化，连接 1800+ 知识单元。

**技术**：Canvas 2D 力导向图（Barnes-Hut O(n log n)）+ Web Worker

**特性**：

- 8 域节点筛选（物理/历史/哲学/生命科学/宇宙学/数学/经济学/心理学）
- BFS 最短路径高亮
- 聚类模式 + 搜索 + 缩略导航
