## 2. `app/layout.tsx`（关键片段）

```tsx
import { spaceGrotesk, plexMono } from "@/lib/fonts";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html
      lang="zh-CN"
      className={`${spaceGrotesk.variable} ${plexMono.variable}`}
      suppressHydrationWarning
    >
      <head>
        {/* CJK + 楷体：Google Fonts。也可自托管 woff2。 */}
        <link rel="preconnect" href="https://fonts.googleapis.com" />
        <link rel="preconnect" href="https://fonts.gstatic.com" crossOrigin="" />
        <link
          href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@300;400;500;700&family=Noto+Serif+SC:wght@400;500;600;700&family=LXGW+WenKai+TC:wght@400;700&display=swap"
          rel="stylesheet"
        />
        {/* 防止暗/亮首屏闪烁（FOUC）：在任何渲染前根据存储值给 <html> 加 .light */}
        <script
          dangerouslySetInnerHTML={{
            __html:
              "(function(){try{var t=localStorage.getItem('uk-theme');if(t==='light')document.documentElement.classList.add('light');}catch(e){}})()",
          }}
        />
      </head>
      <body>{children}</body>
    </html>
  );
}
```

> 同时把 `--font-noto-sans-sc` / `--font-noto-serif-sc` 这种「变量名」从 @theme 栈里换成直接写字体名 `"Noto Sans SC"` / `"Noto Serif SC"`（因为它们由 `<link>` 提供，没有 next/font 变量）。见下方 @theme。

## 5. `DomainCard.tsx` 改造要点（before → after）

**移除**：`domain-card__shimmer`、`__glow`、`__topline`、渐变左栏 rail、`gradient` 顶线、`backdrop-blur-xl`、glow 阴影。

**改为**（结构示意）：

  href={href}

## 6. 主题切换（核对 `ThemeProvider` / `ThemeToggle`）

- 统一 `localStorage` key：`uk-theme`，值 `'dark' | 'light'`。
- 切换时：`document.documentElement.classList.toggle('light', theme==='light')` + 写入 storage。
- 首屏 FOUC 由第 2 节内联脚本兜底。
- 门户与所有领域共用同一机制（领域 root 若是 `.economics-root` 等，确保 `.light` 选择器同时覆盖 `:root` 与 `.xxx-root`，见第 4 节写法）。
