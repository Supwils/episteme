小结 · Synthesis

一场没有胜负的对话。希腊的「知即德」与儒家的「习以成德」，从两个方向逼近同一个问题——前者由「知」通向德，后者由「行」涵养德。两千年后，它们仍各自照亮着「人应如何生活」的不同路径。
