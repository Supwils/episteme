为知识库重新定调：四个方向

现版本是典型的「深空科技」皮肤——霓虹渐变、发光卡片、点阵网格、流光动画，加上 Fraunces + Inter。这套组合很「AI 生成感」：所有东西都在发光、漂浮、渐变。

下面四个方向都拿掉了发光与渐变，回到真正「有设计」的基本功：克制的配色、讲究的字体搭配、清晰的栅格与留白、以及恰到好处的层级。每个方向都用首页（导航 + Hero + 领域入口）来呈现。

01

典籍 — 暖纸编辑风

像一本精心排版的百科典籍。暖纸底、墨黑正文、衬线标题，领域用「目录索引」的方式编号陈列。单一陶土色点缀，无发光、无渐变。安静、耐看、有学术分量。

观测台 — 克制暗色

保留「宇宙」的暗色气质，但纪律严明：纯哑光近黑、无任何发光与渐变，发丝级描边，等宽字体做元数据标签，一抹琥珀色点缀。像天文台的观测日志 / 航天技术文档。深邃但不喧哗。

美术馆 — 色块瑞士风
