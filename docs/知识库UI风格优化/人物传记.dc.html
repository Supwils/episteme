苏格拉底

Socrates

西方哲学的奠基者之一。他不著一字，却用一生的提问，把哲学从仰望星空拉回到「人应当如何生活」。

核心思想 / Key Ideas

Maieutics

产婆术

如同助产士接生婴儿，他通过连续提问，帮助对方「生出」本就潜藏在心中的知识，而非灌输。

Elenchus

诘问法

不断追问对方的定义与前提，暴露其信念中的自相矛盾——真理在反复的辩驳中浮现。

Virtue is Knowledge

知德合一

没有人明知故犯——一切恶行都源于无知。若真正懂得何为善，人必然会行善。

Docta Ignorantia

无知之知

「我知道我一无所知。」自觉无知，正是一切智慧与求知的真正起点。

未经省察的人生不值得过。—— 苏格拉底，于雅典的审判（柏拉图《申辩篇》）

思想来源 / Sources

苏格拉底本人未留下任何著作。我们对他的全部认识，来自三类「二手」记述——这本身就成了哲学史上著名的「苏格拉底问题」：

  • 柏拉图《对话录》——《申辩》《克里同》《斐多》《会饮》等，最具哲学深度，但难辨何处是柏拉图的发挥。
  • 色诺芬《回忆苏格拉底》——更朴实、更贴近日常的苏格拉底。
  • 阿里斯托芬《云》——喜剧中被讽刺的形象，一个反面的旁证。
