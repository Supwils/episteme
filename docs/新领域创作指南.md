## 3. 正文结构（建议骨架，按主题调整）

1. 开篇钩子：一个具体场景/年份/人物切入。
2. `## 破除误解`：常见但过浅的理解。
3. `## 现场 / 核心`：把核心思想或机制讲清楚，为什么重要。
4. `## 结构 / 展开`：分点、对比（鼓励用表格对比不同流派/方法/路线）。
5. `## 代价与争议`：局限、批评、对立观点（务必呈现争论双方）。
6. `## 跨域连接`：连到数学/哲学/历史/经济等，可用 `[[slug]]` 标注同域条目。
7. `## 延伸阅读`：真实文献（作者、书名/期刊、年份）。

深度要求：正文 ≥ 100 非空行（人物/重大概念）或 ≥ 80 行（一般条目）；段落不要写成长墙。

## 4. 事实纪律（最高优先级）

- 凡具体年份/数据/人名/论断，用 WebSearch 核实；**绝不编造**人名、论文、数据、引语。
- 区分"已确证"与"有争议/推测"；在世/已故要核实。
- 引语必须可溯源，否则转述或标"传"。
- 政治内容尤其敏感：**呈现争论各方，不做单方面价值裁决**（遵循知识精神"反对单一叙事"）。对各国政体/意识形态保持学理中立、有来源。

## 5. 语言

中文正文；专有名词、人名英文原名首次出现可并列（如 香农 Claude Shannon）；代码/slug/变量英文。
